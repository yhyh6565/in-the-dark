# 김솔음의 유저 저니 - Decision Tree
## 세광공업고등학교: 어둠은 공포

---

## 📊 전체 플로우차트 개요

```
시작점 (교실에서 눈을 뜸)
│
├─ 명찰 수집 선택
│  ├─ 명찰 0개 → [배드엔딩 0: 저주받은 자]
│  ├─ 명찰 1-9개 → [엔딩 2: 탈출] or [엔딩 3: 클리어]
│  └─ 명찰 10개+ → 상위 엔딩 루트 진입
│
├─ 학생 루트 진입 선택
│  ├─ 진입 안 함 → 제한적 플레이
│  └─ 진입 → 전체 컨텐츠 접근
│
├─ NPC 관계
│  ├─ 이결 구출 → [엔딩 7: 함께] 가능
│  └─ 이결 무시 → [엔딩 7] 불가능
│
├─ 부적 복구
│  ├─ 복구 성공 → [해피엔딩 01: 도서관에 잠든 학교]
│  └─ 복구 안 함 → [해피엔딩 02: 해방의 졸업식]
│
└─ 졸업식 참가
   ├─ 불참 → [엔딩 2 or 3]
   ├─ 실패 → [배드엔딩 05: 학교의 종말]
   └─ 성공 → [해피엔딩 01 or 02 or 7]
```
