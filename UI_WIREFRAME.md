## 1. HUD (인게임)

### 1.1 전체 레이아웃

```
게임 화면 (1920x1080)
┌────────────────────────────────────────────────────────────┐
│  [점멸 타이머 바]          [시간: 02:34]     [배터리🔋 75%] │  ← 상단
│                                                            │
│                                                            │
│                      [게임 화면]                            │
│                     (탑다운 뷰)                            │
│                                                            │
│                    [플레이어 캐릭터]                        │
│                                                            │
│                                                            │
│                                                            │
│  [명찰×2]                                                  │  ← 좌측 하단
│                                                            │
│                              [1][2][3][4][5]  ← 퀵슬롯      │  ← 하단 중앙
│                                                            │
│  [시스템 메시지]                                           │  ← 하단
└────────────────────────────────────────────────────────────┘
```

### 1.2 상단 HUD (우측)

**위치**: 화면 우측 상단
**크기**: 400px × 80px

```
┌──────────────────────────────────┐
│ 점멸까지: [████████░░] 120초     │  ← 타이머 바
│ 배터리:   [██████████] 100%     │  ← 배터리 게이지
│ 시간:     02:34                 │  ← 게임 시간
└──────────────────────────────────┘
```

**상세**:
- **점멸 타이머 바**:
  - 크기: 200px × 20px
  - 색상: 초록(0-90초) → 노랑(90-115초) → 빨강(115-120초)
  - 숫자: 남은 시간 실시간 표시

- **배터리 게이지**:
  - 크기: 200px × 20px
  - 색상: 파랑(50-100%) → 노랑(25-50%) → 빨강(0-25%)
  - 손전등 OFF 시 투명도 50%

- **게임 시간**:
  - 형식: MM:SS
  - 색상: 흰색
  - 폰트: 8비트 픽셀 폰트

### 1.3 좌측 하단 (명찰 카운터)

**위치**: 화면 좌측 하단
**크기**: 120px × 80px

```
┌────────────┐
│  명찰 📛   │
│            │
│  [●][●]   │  ← 2개 보유
│            │
└────────────┘
```

**상세**:
- 명찰 아이콘: 32x32px
- 최대 표시: 5개 (5개 이상은 "+N" 숫자 표시)
- 색상:
  - 1-3개: 초록색 테두리
  - 4개 이상: 빨간색 테두리 (경고)

### 1.4 하단 중앙 (퀵슬롯)

**위치**: 화면 하단 중앙
**크기**: 300px × 60px

```
┌───┬───┬───┬───┬───┐
│ 🔦│ 🔋│   │   │   │  ← 슬롯 5개
└───┴───┴───┴───┴───┘
  1   2   3   4   5    ← 키 번호
```

**상세**:
- 슬롯 크기: 각 50x50px
- 슬롯 간격: 10px
- 선택된 슬롯: 노란색 테두리 (2px)
- 빈 슬롯: 회색 배경
- 아이템 아이콘: 32x32px (슬롯 중앙)

**키 바인딩**:
- 1~5: 슬롯 선택
- Space / 우클릭: 선택된 아이템 사용

### 1.5 하단 (시스템 메시지)

**위치**: 화면 하단 (퀵슬롯 아래)
**크기**: 800px × 100px

```
┌──────────────────────────────────────┐
│                                      │
│  [시스템] 전등이 곧 꺼질 것 같다     │
│                                      │
└──────────────────────────────────────┘
```

**상세**:
- 배경: 반투명 검은색 (80% 투명도)
- 텍스트: 흰색 (경고 시 빨간색)
- 폰트: 8비트 픽셀 폰트, 18px
- 표시 시간: 3초 (자동 페이드 아웃)
- 최대 2줄

## 2. 인벤토리 화면

### 2.1 전체 레이아웃

**트리거**: [Tab] 키
**상태**: 게임 일시정지

```
┌────────────────────────────────────────────────────────┐
│                   인벤토리 📦                          │
│                                                        │
│  ┌──────────────────┐  ┌────────────────────────┐    │
│  │   아이템 목록     │  │   선택된 아이템 정보    │    │
│  ├──────────────────┤  ├────────────────────────┤    │
│  │ [🔦] 비상 손전등  │  │ [🔦]                   │    │
│  │ [🔋] 배터리 x3   │  │ 비상 손전등            │    │
│  │ [📛] 명찰 x2     │  │                        │    │
│  │ [📝] 쪽지        │  │ "다이소에서 팔 법한     │    │
│  │                  │  │  싸구려 노란색 플라스틱│    │
│  │                  │  │  손전등"               │    │
│  │                  │  │                        │    │
│  │                  │  │ 배터리: 10분           │    │
│  │                  │  │ 범위: 10m              │    │
│  │                  │  │                        │    │
│  │                  │  │ [장착]  [버리기]       │    │
│  └──────────────────┘  └────────────────────────┘    │
│                                                        │
│  [Tab - 닫기]  [1~5 - 퀵슬롯 등록]  [Del - 버리기]   │
└────────────────────────────────────────────────────────┘
```

### 2.2 아이템 목록 (좌측)

**크기**: 400px × 600px
**스크롤**: 가능 (아이템 많을 시)

**아이템 표시 형식**:
```
[아이콘] 아이템명 x개수
```

**정렬**:
1. 장비 (손전등 등)
2. 소모품 (배터리, 회복약 등)
3. 명찰
4. 기타 (쪽지, 열쇠 등)

**선택**:
- 마우스 클릭 또는 방향키
- 선택된 아이템: 노란색 배경

### 2.3 아이템 상세 (우측)

**크기**: 500px × 600px

**표시 정보**:
- 아이템 아이콘 (대형, 64x64px)
- 아이템 이름
- 설명 (최대 5줄)
- 수치 정보 (배터리, 범위 등)
- 버튼 (장착, 사용, 버리기)

**버튼**:
- [장착]: 퀵슬롯에 등록
- [사용]: 즉시 사용 (소모품)
- [버리기]: 인벤토리에서 제거 (확인 팝업)

## 3. 대화 시스템

### 3.1 일반 대화창

**위치**: 화면 하단
**크기**: 1600px × 300px

```
┌────────────────────────────────────────────────────────┐
│                                                        │
│  [🧑 이결]                                             │
│                                                        │
│  "으윽... 여긴... 어디야..."                            │
│                                                        │
│                                                        │
│                               [Space - 다음]  [E - 선택]│
└────────────────────────────────────────────────────────┘
```

**상세**:
- 배경: 반투명 검은색 (90%)
- 화자 이름: 좌측 상단 (아이콘 + 이름)
- 텍스트: 흰색, 18px 픽셀 폰트
- 최대 3줄
- 타이핑 효과: 1글자 0.05초

### 3.2 선택지 대화창

```
┌────────────────────────────────────────────────────────┐
│  [🧑 이결]                                             │
│  "너... 누구야?"                                        │
│                                                        │
│  ┌─────────────────────────┐                          │
│  │ ▶ "나는 김솔음이야."     │  ← 선택 1                │
│  │   "도와줄게."            │  ← 선택 2                │
│  │   "..."                  │  ← 선택 3                │
│  └─────────────────────────┘                          │
│                                                        │
│                  [방향키 - 이동]  [E - 선택]           │
└────────────────────────────────────────────────────────┘
```

**선택지**:
- 최대 4개
- 선택된 선택지: ▶ 표시
- 방향키 또는 마우스로 선택

### 3.3 필담 (전학생 모드)

```
┌────────────────────────────────────────────────────────┐
│                     [노트 종이]                         │
│                                                        │
│  (NPC가 쓴 글씨가 나타난다...)                         │
│                                                        │
│  "노루 씨?? 지금 자아가 있으신가요?"                    │
│                                                        │
│  (당신의 대답을 선택하세요)                            │
│                                                        │
│  ┌───────────────┐ ┌───────────────┐                 │
│  │ [네, 있어요]   │ │ [아니요]       │                 │
│  └───────────────┘ └───────────────┘                 │
│  ┌───────────────┐                                    │
│  │ [모르겠어요]   │                                    │
│  └───────────────┘                                    │
└────────────────────────────────────────────────────────┘
```

**특징**:
- 화면 전체가 노트 종이 배경
- 상대방 글씨: 실시간 타이핑 효과
- 선택지: 버튼 형태 (CTA)
- **주의**: 백일몽 주식회사 직원(장허운, 고영은, 백사헌)은 대화 불가능 ('침묵의 가이드' 방식)

## 8. UI 스타일 가이드

### 8.1 색상 팔레트

**기본 색상**:
- 배경: `#1a1a1a` (검은색)
- 텍스트: `#f0f0f0` (흰색)
- 테두리: `#808080` (회색)

**강조 색상**:
- 선택: `#ffff00` (노란색)
