## 엔딩 개요

### 기본 정보
- **엔딩 타입**: 해피엔딩 (True Ending)
- **결말 요약**: 학교가 한빛도서관의 '비공개 코너'로 안전하게 격리/보존됨
- **엔딩 장면**: 5월의 따뜻한 햇살 아래 모든 학생과 요원들이 낮잠을 자는 평화로운 모습
- **스코어**: 표기 안 됨 (A급 이상 추정)
- **크레딧 메시지**:
  ```
  아이들은 다시 안전히 구조를 기다리고 있습니다.
  찾아봐주세요.
  ```

### 핵심 테마
- **협력**: 재난관리국 요원 3명(청동, 최요원, 해금) + 백일몽 요원(이자헌)과의 팀워크
- **봉인**: 한빛도서관 통로에 완성된 부적으로 학교를 안전하게 격리
- **평화**: 학생들이 위험에서 벗어나 안전하게 구조를 기다림

## 진입 조건

1. **전학생 모드 진입**
   - 이결에게 이불 덮어주기 완료
   - 세광공고 교복 획득 및 착용

2. **뒤뜰 발견**
   - 전학생 모드 상태에서 3층 양호실 창문 상호작용
   - 창문을 통해 뒤뜰로 진입

3. **핵심 아이템 획득**
   - [부서진 유리 초롱] + [찢어진 부적] 획득 (뒤뜰 화단 흙더미)
   - 이 아이템을 획득하는 순간 **해피엔딩 트리거** 발동

4. **동료 협력**
   - 청동 요원과 협력 (게임 시작부터 가능)
   - 최요원 각성 (해피엔딩 트리거 발동 시 2학년 3반)
   - 해금 요원 각성 (해피엔딩 트리거 발동 시 2학년 3반)

5. **부적 봉인 완성**
   - 한빛도서관 통로(도서실 소화전)에 완성된 부적 설치

## 핵심 아이템

### 1. 부서진 유리 초롱 + 찢어진 부적
- **위치**: 뒤뜰 화단 흙더미 속
- **획득 조건**: 전학생 모드 착용 상태에서 뒤뜰 진입
- **역할**: 해피엔딩 01 트리거 발동 아이템

### 2. 도서대출표 부적
- **제공자**: 최요원 (재난관리국)
- **설명**: "이걸 가져가. 팀장님이... 남기신 거야." (박홍림 팀장의 유산)
- **역할**: 뒤뜰에 파묻어 부적의 일부로 사용

### 3. 유리 초롱
- **제공자**: 해금 요원 (재난관리국)
- **외형**: 푸른 빛 나는 유리초롱
- **역할**: 부적을 완성하는 핵심 부품

### 4. 도깨비불
- **획득 시점**: 양호실 재회 이벤트 (청동 요원 + 최요원 + 김솔음)
- **연출**: 3명을 둘러싼 신묘한 푸른 불빛이 번쩍임 (3초간)
- **역할**: 뒤뜰에서 사용하여 부적 발동 시작

## 엔딩 시퀀스

### Phase 0: 프롤로그 - 각성

> **원작 출처**: 도깨비불 형성 장면 (최요원, 청동 요원, 김솔음 각성)
> **게임 타이밍**: 해피엔딩 루트 진입 전 선행 이벤트

#### 0-1. 동시 각성 이벤트

**트리거 조건**:
- 이결에게 이불 덮어주기 완료
- 세광공고 교복 획득
- 특정 게임 진행도 도달 (예: 1층 탐험 완료)

**연출 시퀀스** (15초):
1. **화면 암전** (2초)
   - BGM 페이드아웃

2. **컷신 시작** - 3층 양호실
   - 카메라: 양호실 천장을 올려다보는 POV
   - 김솔음이 양호실 침대에서 눈을 뜬다

3. **동시 각성 연출** (5초)
   ```
   [시스템] 청동 요원이 눈을 떴습니다
   [시스템] 최요원이 눈을 떴습니다
   ```
   - 카메라: 양호실 전체를 보여줌
   - 비주얼: 3명이 각각 다른 침대에서 동시에 일어남

4. **도깨비불 형성** (3초)
   - 3명 주변에 **푸른 불빛**이 서서히 떠오름
   - 불빛들이 서로를 향해 끌려가듯 모임
   - 하나의 **신묘한 푸른 구체**로 합쳐짐

5. **시스템 메시지** (3초)
   ```
   [시스템] 아이템 획득: [도깨비불의 씨앗]
   세 사람의 의지가 하나로 모였습니다.
   ```

#### 0-2. 첫 대화

**청동 요원**: "...다들 괜찮으신가요?"

**최요원**: "네. 저는 괜찮습니다. 근데... 이게 뭐죠?" (푸른 불빛을 바라보며)

**김솔음**: (선택지)
- → "우리... 왜 같이 깨어난 거죠?"
- → "이 불빛... 따뜻해요."

**청동 요원**: "...중요한 일이 일어날 것 같습니다. 조심하세요."

**Phase 0 종료**
- 플레이어에게 컨트롤 복귀
- [도깨비불의 씨앗] 인벤토리에 추가됨
- 이후 Phase 1 (뒤뜰 발견) 진행 가능

> **설계 노트**:
> - Phase 0는 해피엔딩 루트의 "예고편" 역할
> - 플레이어에게 "이 세 사람이 중요하다"는 암시 제공
> - 도깨비불의 씨앗은 Phase 3에서 완전한 [도깨비불]로 각성

### Phase 1: 해피엔딩 트리거 발동 (뒤뜰)

#### 1. 뒤뜰 진입 및 발견
- **조건**: 전학생 모드 착용 상태에서 3층 양호실 창문 상호작용
- **행동**: 창문을 통해 뒤뜰로 내려감
- **발견**: 화단 흙더미를 파헤쳐 **[부서진 유리초롱]** + **[찢어진 부적]** 획득

#### 2. 동시다발 시스템 메시지
- **트리거**: 유리초롱/부적 획득 즉시
- **시스템 메시지** (연속 출력):
  ```
  [시스템] 청동 요원이 기억을 찾았습니다
  [시스템] 최요원이 2학년 3반에서 눈을 떴습니다
  [시스템] 해금 요원이 2학년 3반에서 눈을 떴습니다
  [시스템] 이결이 손바닥을 내려다봅니다
  ```
- **효과**: BGM 변화 (불안한 분위기 시작)

### Phase 1.5: 부적 실패 이벤트

> **원작 출처**: 부적 실패 후 절망→희망 전환 아크
> **핵심 테마**: Hope → Despair → Recovery (감정 롤러코스터)
> **게임 타이밍**: Phase 1 직후, Phase 2 전

#### 1.5-1. 부적 발동 시도 (뒤뜰)

**상황**:
- 김솔음이 [부서진 유리초롱] + [찢어진 부적]을 획득한 직후
- 뒤뜰 화단 앞에 서 있음

**김솔음의 내면 독백**:
```
김솔음: (내면) 이게... 부적이다.
김솔음: (내면) 이걸로 학교를 봉인할 수 있을지도 몰라.
```

**행동**:
- 플레이어가 [찢어진 부적]을 사용 시도
- 자동 트리거: 부적을 화단에 파묻으려 시도

#### 1.5-2. 실패 연출 (3초)

**비주얼**:
- 부적이 희미하게 빛나다가... **스르륵 사그라짐**
- 아무 일도 일어나지 않음

**시스템 메시지**:
```
[시스템] 부적이 반응하지 않습니다
[시스템] 무언가 부족합니다
```

**김솔음 대사**:
```
김솔음: "안 돼... 왜 안 되는 거지?"
김솔음: "부서진 유리 초롱만으로는... 부족한 건가?"
```

#### 1.5-3. 절망 타이밍 (5초)

**화면 효과**:
- 화면이 서서히 어두워짐 (비네팅 효과 강화)
- BGM이 불안한 톤으로 전환

**김솔음 독백**:
```
김솔음: (내면) 이대로는... 학교에서 벗어날 수 없어.
김솔음: (내면) 다른 방법이 있을까...?
```

**타이밍**:
- 5초간 플레이어에게 "막막함"을 느끼게 함
- **이 순간이 감정적 저점**

#### 1.5-4. 희망의 전환점 (Phase 2로 연결)

**시스템 메시지** (5초 후):
```
[시스템] 누군가 당신을 찾고 있습니다
[시스템] 청동 요원과 최요원이 3층 양호실로 달려옵니다
```

**김솔음 반응**:
```
김솔음: "...!"
김솔음: "양호실로 가야 해!"
```

**플레이어 액션**:
- 뒤뜰에서 양호실로 복귀 (창문 등반)
- **Phase 2로 자동 진행**

> **설계 노트**:
> - Phase 1.5는 "부적만으로는 부족하다"는 깨달음을 줌
> - 이후 Phase 2-7에서 "도깨비불", "도서대출표 부적", "유리 초롱"을 모으는 여정의 동기 부여
> - 감정 아크: 희망(부적 발견) → 절망(실패) → 재희망(동료 집결)

### Phase 2: 양호실 재회 (3층)

- **유저 행동**: 뒤뜰에서 양호실 창문을 통해 복귀

#### 4. 양호실 재회 이벤트
- **타이밍**: 양호실 복귀 5초 후
- **연출**: 양호실 문이 열리며 **청동 요원**, **최요원** 입장
- **컷신**:

- **보상**:
  ```

  ```

### Phase 3: 부적 준비 (양호실 → 뒤뜰)

#### 5. 도서대출표 부적 획득
- **행동**: 최요원에게 다가가 상호작용 [E]
- **최요원 대사**: "이걸 가져가. 팀장님이... 남기신 거야."
- **아이템 획득**: **[도서대출표 부적]**

#### 6. 뒤뜰 부적 파묻기
- **이동**: 양호실 창문 → 뒤뜰로 재진입
- **제약**: 재난관리국 요원 3명은 뒤뜰로 따라 내려올 수 없음 (뒤뜰이 보이지 않음)
- **행동**: 화단에 **[도서대출표 부적]** 파묻기
- **완료**: 흙더미를 다시 덮음

#### 7. 도깨비불 발동
- **행동**: 인벤토리에서 **[도깨비불]** 사용
- **효과**: 화면 전체에 푸른 불빛이 퍼짐
- **타이밍**: 도깨비불 사용 5초 후
- **시스템 경고 메시지**:
  ```
  [시스템] 숨박꼭질끝났다놀이끝났다
  침입했다침입했다학교에먹는다막는다뒤에뒤에뒤에위험해사라진다죽어안돼제발안돼제물아니야침입했다침입했다비명어이구슬퍼먹는다저기어둠에서나온것이밀려온다삭윈것이먹어치운
  ```
- **연출**: 화면 진동, 붉은 플래시 효과

### Phase 3.5: 시스템 공지 공포 연출

> **원작 출처**: "숨박꼭질끝났다" 시스템 폭주 장면
> **핵심 테마**: 학교 시스템의 각성 및 반격
> **게임 타이밍**: 도깨비불 발동 직후 (Phase 3-7 확장)
> **지속 시간**: 약 15초

#### 3.5-1. 시스템 오류 전조 (3초)

**트리거**:
- Phase 3의 도깨비불 발동 직후
- 푸른 불빛이 화면 전체로 퍼진 직후

**비주얼 효과**:
- 화면이 1초간 **순백색**으로 플래시
- 이후 모든 UI가 **깜빡이기 시작**
- 텍스트 폰트가 불안정하게 떨림

**사운드**:
- 고음의 **이명 소리** (Tinnitus effect, 1초)
- BGM 급격히 증폭 후 끊김

#### 3.5-2. 시스템 메시지 폭주 (5초)

**화면 연출**:
- 화면 전체에 텍스트가 **중첩되어 쏟아짐**
- 읽을 수 없을 정도로 빠른 속도
- 텍스트 색상: 붉은색 → 검은색 → 깜빡임

**시스템 메시지** (원작 그대로, 띄어쓰기 없음):
```
[시스템] 숨박꼭질끝났다놀이끝났다
침입했다침입했다학교에먹는다막는다뒤에뒤에뒤에위험해사라진다죽어안돼제발안돼제물아니야침입했다침입했다비명어이구슬퍼먹는다저기어둠에서나온것이밀려온다삭윈것이먹어치운
```

**비주얼 효과**:
- 메시지가 화면을 **상하좌우로 스크롤**
- 일부 단어만 크게 강조:
  - **"침입했다"** (붉은색, 2배 크기)
  - **"먹는다"** (검은색, 깜빡임)
  - **"죽어"** (붉은색, 화면 중앙에 1초간 고정)

#### 3.5-3. 화면 진동 및 공포 연출 (5초)

**카메라 효과**:
- 화면이 **격렬하게 진동** (Screenshake, Intensity: 10)
- 진동 패턴: 불규칙 (0.1초 간격으로 랜덤)

**비주얼 효과**:
- **붉은 플래시** (화면 전체, 0.5초 간격으로 3회)
- 화면 가장자리에서 **검은 촉수 같은 형상**이 중앙으로 뻗어옴
- 뒤뜰 배경이 일그러지며 왜곡됨

**사운드**:
- 불쾌한 **긁는 소리** (Metal scraping)
- 먼 곳에서 들려오는 **비명 소리** (Distant screams, 8bit distorted)
- 심장 박동 소리 (Heartbeat, 증폭)

#### 3.5-4. 침묵과 여운 (2초)

**연출**:
- 모든 소리와 효과가 **갑자기 멈춤**
- 2초간 **완전한 정적** (침묵)
- 화면은 정상으로 복귀하지만 약간 어둡게 유지

**김솔음 독백**:
```
김솔음: (내면) ...무슨... 일이...?
```

**시스템 메시지** (작게, 천천히):
```
[시스템] 당신을 인식했습니다
```

**Phase 3.5 종료**
- 플레이어에게 컨트롤 복귀
- 뒤뜰 → 양호실로 복귀 가능
- **Phase 4로 진행**

### Phase 6.5: 강당 전투 & 사인검 QTE

> **원작 출처**: 해금 요원의 사인검 전투 장면
> **핵심 메커니즘**: 5-Stage QTE + 검결(劍訣) 낭독 시스템
> **게임 타이밍**: 5층 강당 도착 후, 선생님 추격 중
> **지속 시간**: 약 45초 (QTE 포함)

#### 6.5-1. 5층 강당 앞 도착

**상황**:
- 김솔음, 청동 요원, 최요원이 5층 강당 복도 도착
- 뒤에서 **선생님**(시체골렘)이 추격 중
- 강당 문이 굳게 닫혀 있음

**비주얼**:
- 복도 끝에서 선생님의 거대한 그림자가 다가옴
- 강당 문 손잡이에 **부서진 유리초롱**과 **찢어진 부적**이 걸려 있음 (희미하게 빛남)
- 화면 하단: "거리: 15m... 12m... 10m..."

**시스템 메시지**:
```
[시스템] 강당 문이 열리지 않습니다
[시스템] 선생님이 접근하고 있습니다
```

#### 6.5-2. 해금 요원 등장 (5초)

**연출**:
- 갑자기 **2학년 3반 복도에서** 해금 요원이 달려옴
- 해금 요원이 플레이어 일행과 선생님 사이에 **착지**

**해금 요원 대사**:
```
해금: "여기는 내가 막을게!"
해금: "강당 문을 열어! 빨리!"
```

**비주얼**:
- 해금 요원이 **사인검**(Sign Sword)을 꺼냄
- 사인검에서 푸른 불꽃이 타오름
- 카메라: 해금 요원과 선생님의 대치 구도

#### 6.5-3. 사인검 QTE 시퀀스 (30초)

> **QTE 시스템 설명**:
> - 화면에 **검결(劍訣)** 텍스트가 나타남
> - 플레이어는 해당 한자/한글을 **순서대로 입력** (버튼 매핑: A, B, X, Y, L, R)
> - 성공 시: 해금 요원의 검격이 성공, 선생님 밀쳐냄
> - 실패 시: 해금 요원이 피격, 플레이어에게 2초 패널티

**QTE Stage 1** (5초) - 초식: 파사참(破邪斬)
- **검결**: "사(邪) - 기(氣) - 파(破)"
- **버튼**: A → B → X
- **성공 연출**: 사인검이 수평으로 베임, 선생님이 3m 뒤로 밀려남
- **해금 대사**: "첫 번째!"

**QTE Stage 2** (5초) - 초식: 정화섬(淨化閃)
- **검결**: "정(淨) - 화(化) - 섬(閃)"
- **버튼**: B → X → Y
- **성공 연출**: 사인검이 수직으로 내려침, 푸른 빛 폭발
- **해금 대사**: "두 번째!"

**QTE Stage 3** (7초) - 초식: 진언참(眞言斬)
- **검결**: "진(眞) - 언(言) - 부(符) - 참(斬)"
- **버튼**: A → X → B → Y
- **성공 연출**: 사인검이 X자로 베임, 선생님의 팔 일부가 떨어짐
- **해금 대사**: "세 번째!"
- **선생님 반응**: 으르렁거리는 소리, 더 빠르게 접근

**QTE Stage 4** (7초) - 초식: 도깨비불 인(印)
- **검결**: "도(道) - 화(火) - 인(印) - 결(結)"
- **버튼**: L → R → A → B
- **성공 연출**: 사인검 끝에서 도깨비불이 발사됨, 선생님 상체에 명중
- **해금 대사**: "네 번째... 거의 다 왔어!"

**QTE Stage 5** (6초) - 초식: 최종 봉인격 (Final Seal Strike)
- **검결**: "봉(封) - 인(印) - 완(完) - 성(成) - 참(斬)"
- **버튼**: A → B → X → Y → A (5연타)
- **난이도**: 가장 높음 (입력 시간 제한: 각 0.8초)
- **성공 연출**:
  - 사인검이 **거대한 부적 형상**으로 변화
  - 해금 요원이 검을 선생님에게 찌름
  - 선생님이 **푸른 결계**에 갇혀 움직임 정지 (10초간)
- **해금 대사**: "지금이야! 문을 열어!"

#### 6.5-4. 강당 문 개방 (3초)

**자동 진행**:
- 선생님이 봉인된 동안
- 청동 요원이 강당 문 손잡이의 부적을 만짐
- **강당 문이 스르륵 열림**

**시스템 메시지**:
```
[시스템] 강당 문이 열렸습니다
[시스템] 서둘러 들어가세요!
```

**해금 요원 대사**:
```
해금: "빨리! 봉인이 10초밖에 안 가!"
```

#### 6.5-5. 강당 진입 및 Phase 종료

**연출**:
- 플레이어 일행(김솔음, 청동, 최요원) 강당 안으로 진입
- 해금 요원도 뒤따라 들어옴
- **강당 문이 닫힘** (쾅!)
- 문 밖에서 선생님의 주먹이 문을 두드리는 소리 (쿵! 쿵!)

**카메라**:
- 강당 내부 전경
- 조명이 깜빡이는 5층 강당
- **졸업식 단상**이 보임

**시스템 메시지**:
```
[시스템] 강당에 진입했습니다
[시스템] 졸업식이 곧 시작됩니다
```

**Phase 6.5 종료**
- Phase 7로 진행 (2학년 3반에서 이자헌 과장 조우는 이미 완료되었으므로, Phase 7 스킵 가능)
- 또는 Phase 7을 강당 내부 이벤트로 재구성

> **설계 노트**:
> - QTE 난이도 조절 가능: Easy 모드에서는 입력 시간 +50%
> - QTE 실패 시에도 이야기는 진행되지만, 해금 요원이 부상 → 이후 엔딩 크레딧에서 붕대 감은 모습
> - 사인검 검결은 원작의 "부정한 것을 파괴하는 칼" 컨셉 반영
> - 45초 QTE는 긴장감 유지를 위한 최적 시간

### Phase 8: 도서실 봉인 & 이결 희생

> **원작 출처**: 이결 희생 시퀀스 전문 (NOVEL_TRANSCRIPT_SAEKWANG_HAPPYENDING.md)
> **핵심 테마**: 희생과 해방 - "넌 이 학교 학생 아니야"
> **게임 타이밍**: 졸업식 성공 직후, 한빛도서관 통로 봉인 시
> **지속 시간**: 약 90초 (60초 컷신 + 30초 인터랙션)

#### 8-1. 한빛도서관 통로 발견 (도서실 소화전)

**상황**:
- 졸업식을 성공적으로 완료 (40초 생존 성공)
- 강당에서 나온 김솔음 일행
- 청동 요원이 부적을 완성한 상태

**청동 요원 대사**:
```
청동: "이제 부적을 한빛도서관 통로에 설치해야 합니다."
청동: "도서실로 가시죠."
```

**이동**:
- 5층 → 4층 → 3층 도서실
- 도서실 내부의 **소화전** 앞에 도착

**비주얼**:
- 소화전 문이 희미하게 빛남 (푸른 빛)
- 소화전 주변에 **부적 설치 자리**가 표시됨

#### 8-2. 부적 설치 준비 (5초)

**청동 요원 행동**:
- 청동 요원이 완성된 부적을 꺼냄
- 부적 구성:

## 개발 시 주의사항

### 복잡도 높은 요소
1. **다중 NPC AI**: 3명의 요원이 플레이어를 따라다니며 협력
2. **동시다발 이벤트**: 여러 장소에서 NPC가 동시에 각성
3. **복잡한 시퀀스**: 7개 Phase로 구성된 긴 이벤트 체인
4. **조건부 트리거**: 정확한 타이밍과 순서가 중요
5. **특수 연출**: 푸른 불빛, 살점 번짐, 암전 등 다양한 비주얼 효과

- 전학생 모드 시스템 (교복 착용)
- 뒤뜰 히든 맵

- 엔딩 크레딧 시스템
