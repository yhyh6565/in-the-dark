## 엔딩 개요

### 기본 정보
- **엔딩 타입**: 해피엔딩 (True Ending)
- **결말 요약**: 학교가 한빛도서관의 '비공개 코너'로 안전하게 격리/보존됨
- **엔딩 장면**: 5월의 따뜻한 햇살 아래 모든 학생과 요원들이 낮잠을 자는 평화로운 모습
- **스코어**: 표기 안 됨 (A급 이상 추정)
- **크레딧 메시지**:
  ```
  아이들은 다시 안전히 구조를 기다리고 있습니다.
  찾아봐주세요.
  ```

### 핵심 테마
- **협력**: 재난관리국 요원 3명(청동, 최요원, 해금) + 백일몽 요원(이자헌)과의 팀워크
- **봉인**: 한빛도서관 통로에 완성된 부적으로 학교를 안전하게 격리
- **평화**: 학생들이 위험에서 벗어나 안전하게 구조를 기다림

## 진입 조건

### 필수 조건 (All Required)
1. **전학생 모드 진입**
   - 이결에게 이불 덮어주기 완료
   - 세광공고 교복 획득 및 착용

2. **뒤뜰 발견**
   - 전학생 모드 상태에서 3층 양호실 창문 상호작용
   - 창문을 통해 뒤뜰로 진입

3. **핵심 아이템 획득**
   - [부서진 유리 초롱] + [찢어진 부적] 획득 (뒤뜰 화단 흙더미)
   - 이 아이템을 획득하는 순간 **해피엔딩 트리거** 발동

4. **동료 협력**
   - 청동 요원과 협력 (게임 시작부터 가능)
   - 최요원 각성 (해피엔딩 트리거 발동 시 2학년 3반)
   - 해금 요원 각성 (해피엔딩 트리거 발동 시 2학년 3반)

5. **부적 봉인 완성**
   - 한빛도서관 통로(도서실 소화전)에 완성된 부적 설치

## 핵심 아이템

### 1. 부서진 유리 초롱 + 찢어진 부적
- **위치**: 뒤뜰 화단 흙더미 속
- **획득 조건**: 전학생 모드 착용 상태에서 뒤뜰 진입
- **역할**: 해피엔딩 01 트리거 발동 아이템

### 2. 도서대출표 부적
- **제공자**: 최요원 (재난관리국)
- **설명**: "이걸 가져가. 팀장님이... 남기신 거야." (박홍림 팀장의 유산)

- **제공자**: 해금 요원 (재난관리국)

- **역할**: 부적을 완성하는 핵심 부품

### 4. 도깨비불
- **획득 시점**: 양호실 재회 이벤트 (청동 요원 + 최요원 + 김솔음)
- **연출**: 3명을 둘러싼 신묘한 푸른 불빛이 번쩍임 (3초간)
- **역할**: 뒤뜰에서 사용하여 부적 발동 시작

## 엔딩 시퀀스

### Phase 1: 해피엔딩 트리거 발동 (뒤뜰)

#### 1. 뒤뜰 진입 및 발견
- **조건**: 전학생 모드 착용 상태에서 3층 양호실 창문 상호작용
- **행동**: 창문을 통해 뒤뜰로 내려감
- **발견**: 화단 흙더미를 파헤쳐 **[부서진 유리초롱]** + **[찢어진 부적]** 획득

#### 2. 동시다발 시스템 메시지
- **트리거**: 유리초롱/부적 획득 즉시
- **시스템 메시지** (연속 출력):
  ```
  [시스템] 청동 요원이 기억을 찾았습니다
  [시스템] 최요원이 2학년 3반에서 눈을 떴습니다
  [시스템] 해금 요원이 2학년 3반에서 눈을 떴습니다
  [시스템] 이결이 손바닥을 내려다봅니다
  ```
- **효과**: BGM 변화 (불안한 분위기 시작)

### Phase 2: 양호실 재회 (3층)

#### 3. 요원 집결 예고
- **타이밍**: 시스템 메시지 출력 5초 후
- **메시지**:
  ```
  [시스템] 청동 요원과 최요원이 3층 양호실로 달려옵니다
  ```
- **유저 행동**: 뒤뜰에서 양호실 창문을 통해 복귀

#### 4. 양호실 재회 이벤트
- **타이밍**: 양호실 복귀 5초 후
- **연출**: 양호실 문이 열리며 **청동 요원**, **최요원** 입장
- **컷신**:
  - 3명(김솔음, 청동, 최요원)을 둘러싼 신묘한 푸른 불빛이 번쩍임 (3초간)
  - 푸른 불빛 사라짐
- **보상**:
  ```
  [시스템] 아이템 획득: [도깨비불]
  인벤토리에 '도깨비불'이 추가되었습니다.
  ```

#### 5. 도서대출표 부적 획득

### Phase 4: 유리초롱의 필요성 (양호실)

#### 8. 양호실 복귀

  김솔음: (내면) 유리 초롱이 필요하다.
