# 📖 튜토리얼 플로우차트
## Tutorial Flowchart - 검은 그늘 속에서

> **목적**: 플레이어가 첫 10분 안에 핵심 메커니즘 학습
> **위치**: 2층 1-5반 교실 (시작 지점)
> **예상 소요 시간**: 8-12분

---

## 📋 튜토리얼 목표

플레이어가 학습해야 할 핵심 메커니즘:
1. ✅ 기본 조작 (이동, 상호작용)
2. ✅ 손전등 사용
3. ✅ 시선 시스템 (학생 개체와의 첫 조우)
4. ✅ 점멸 시스템 (첫 점멸 경험)
5. ✅ 은신 (사물함, 책상 밑)
6. ✅ 명찰 시스템
7. ✅ 안전지대 (양호실)

---

## 🎯 튜토리얼 플로우

### Phase 0: 게임 시작 (0분)

**타이틀 화면**:
```
┌────────────────────────────┐
│   검은 그늘 속에서          │
│   In the Dark              │
│                            │
│   [새 게임]                │
│   [불러오기]               │
│   [설정]                   │
│   [종료]                   │
└────────────────────────────┘
```

**[새 게임] 클릭 → 프롤로그 컷신**:
- 메타게임 접속 화면
- 텍스트 파일 읽는 연출
- 페이드 아웃 → 게임 시작

---

### Phase 1: 깨어남 (0-1분)

**Step 1.1: 시작 위치**
- 위치: 2층 1-5반 교실, 김솔음 책상
- 카메라: 페이드 인
- 상태: 플레이어 조작 가능

**독백 (자동)**:
```
김솔음: (내면) '여기는... 어디지?'
김솔음: (내면) '교실...? 내가 왜 여기 있는 거야?'
```

**UI 가이드 (화면 하단)**:
```
[WASD] 키로 이동해보세요
```

**Step 1.2: 첫 이동**
- 플레이어가 WASD 입력
- 김솔음 캐릭터 이동 시작
- UI 가이드 사라짐

**Step 1.3: 손전등 발견**
- 위치: 김솔음 책상 위 (빛나는 아이템)
- UI 가이드:
  ```
  [E] 키로 상호작용
  ```
- [E] 입력 → 손전등 획득

**시스템 메시지**:
```
[아이템 획득] 비상 손전등
배터리: 10분
```

**독백**:
```
김솔음: (내면) '손전등? 왜 이런 게 책상 위에...'
```

---

### Phase 2: 첫 조사 (1-2분)

**Step 2.1: 쪽지 발견**
- UI 가이드:
  ```
  손전등을 조사해보세요 [E]
  ```
- 손전등 상호작용 → 1인칭 뷰 팝업

**1인칭 팝업 화면**:
```
┌────────────────────────────┐
│                            │
│   [손전등 배터리 커버]      │
│                            │
│   안쪽에 쪽지가 보인다     │
│                            │
│   [열어보기]  [닫기]       │
└────────────────────────────┘
```

**[열어보기] 클릭**:
```
┌────────────────────────────┐
│                            │
│   구겨진 쪽지               │
│                            │
│   "빚"                     │
│   (붉은 잉크, 피처럼)      │
│                            │
│   [닫기]                   │
└────────────────────────────┘
```

**독백**:
```
김솔음: (내면) '...빚?'
김솔음: (내면) '이게 무슨 의미지...'
```

**Step 2.2: 손전등 사용 튜토리얼**
- UI 가이드:
  ```
  [F] 키로 손전등 켜기/끄기
  마우스로 방향 조절
  ```
- [F] 입력 → 손전등 ON
- 화면 밝아짐 (Cone 45도)

---

### Phase 3: 환경 탐색 (2-3분)

**Step 3.1: 교실 둘러보기**
- 플레이어가 교실 내부 탐색
- 칠판, 책상, 사물함 조사 가능
- 아이템 없음 (튜토리얼 단순화)

**Step 3.2: 문으로 이동**
- 교실 문 앞 도착
- UI 가이드:
  ```
  [E] 키로 문 열기
  ```
- [E] 입력 → 문 개방 (삐걱 소리)

**Step 3.3: 복도 진입**
- 복도로 나감
- 카메라: 복도 전체 보임
- 분위기: 어둡고 조용함

**독백**:
```
김솔음: (내면) '...너무 조용해.'
김솔음: (내면) '사람이 한 명도 안 보여.'
```

---

### Phase 4: 첫 조우 (3-4분)

**Step 4.1: 학생 개체 출현**
- 위치: 복도 끝 (10타일 거리)
- 외형: 얼굴 없는 학생 (교복)
- 행동: 느리게 배회 (플레이어 방향)

**시스템 메시지 (경고)**:
```
[시스템] 무언가 다가온다
```

**독백 (긴장)**:
```
김솔음: (내면) '저건... 뭐지?'
김솔음: (내면) '얼굴이... 없어.'
```

**Step 4.2: 시선 메커니즘 튜토리얼**
- UI 가이드 (강제):
  ```
  ⚠️ 위험!

  [마우스]로 학생을 향해 시선을 두세요
  [F] 키로 손전등을 켜면 멈춥니다

  ❌ 시선 가이드라인이 닿으면 추격당합니다
  ```

**분기 A: 플레이어가 손전등을 켬**
- 학생 개체 정지
- UI 가이드:
  ```
  ✅ 손전등 빛에 멈췄습니다!
  빛을 유지한 채 옆으로 우회하세요
  ```
- 플레이어가 옆으로 이동
- 안전하게 통과

**분기 B: 플레이어가 시선 가이드라인을 학생에게 닿게 함**
- 학생 개체 추격 시작 (빠른 속도)
- UI 가이드 (긴급):
  ```
  ⚠️⚠️⚠️ 추격당하고 있습니다!

  [Shift]로 달리기
  [F]로 손전등을 켜서 멈추세요
  ```
- 플레이어가 [F] 입력 → 학생 정지
- 안전 확보

**분기 C: 플레이어가 접촉**
- 즉사
- 게임 오버 화면:
  ```
  ┌────────────────────────────┐
  │   사망했습니다              │
  │                            │
  │   [다시 시작]              │
  │   [타이틀로]               │
  └────────────────────────────┘
  ```
- [다시 시작] → Phase 4.1부터 재개

---

### Phase 5: 점멸 경고 (4-5분)

**Step 5.1: 점멸 예고 (5분 경과)**
- 화면 깜빡임 시작 (0.5초 간격)
- **전체 공지 (화면 중앙)**:
  ```
  [전체 공지]
  학생 여러분. 이제 난방과 안전을 위해
  자동 커튼을 닫고 점등하겠습니다.
  ```

**시스템 메시지 (경고)**:
```
[시스템] 전등이 곧 꺼질 것 같다, 숨어야 한다
```

**UI 가이드 (강제, 화면 중앙)**:
```
⚠️⚠️⚠️ 점멸 경고!

5초 후 전등이 깜빡입니다
학생들이 순간이동합니다

📦 사물함이나 책상 밑에 숨으세요!

[Ctrl] - 앉기
[E] - 사물함 진입
```

**카운트다운 UI** (화면 상단):
```
점멸까지: 5... 4... 3... 2... 1...
```

**Step 5.2: 은신 튜토리얼**
- 플레이어가 근처 사물함 발견
- UI 가이드:
  ```
  [E] 키로 사물함에 숨기
  ```
- [E] 입력 → 사물함 진입
- 화면: 사물함 내부 1인칭 뷰 (어둠)

---

### Phase 6: 첫 점멸 (5분)

**Step 6.1: 점멸 발생**
- 타이밍: 게임 시작 5분 후
- 지속: 1초
- 효과: 완전 암전

**화면 연출**:
```
┌────────────────────────────┐
│                            │
│         [완전 어둠]         │
│                            │
│    (스윽- 소리)            │
│    (학생 이동 소리)        │
│                            │
└────────────────────────────┘
```

**Step 6.2: 점멸 종료**
- 1초 후 화면 밝아짐
- 플레이어 위치: 사물함 내부 (안전)

**시스템 메시지**:
```
[시스템] 점멸이 끝났습니다
```

**Step 6.3: 사물함에서 나오기**
- UI 가이드:
  ```
  [E] 키로 사물함에서 나오기
  ```
- [E] 입력 → 사물함 퇴장
- 복도로 복귀

**독백**:
```
김솔음: (내면) '...무사했어.'
김솔음: (내면) '앞으로는 점멸 전에 미리 숨어야겠어.'
```

**Step 6.4: 학생 재배치 확인**
- 복도에 학생 개체 위치 변경됨
- 시스템 메시지 (선택):
  ```
  [시스템] 근처에 무언가 느껴진다
  ```
- 플레이어 긴장감 유도

---

### Phase 7: 명찰 튜토리얼 (6-8분)

**Step 7.1: 첫 전투 (선택)**
- 플레이어가 학생 개체 제압 시도 (자율)
- UI 가이드 (근접 시):
  ```
  [좌클릭] - 공격
  학생을 5회 공격하여 제압
  ```

**전투 시퀀스**:
1. 공격 1회 → 데미지 20
2. 공격 5회 → 학생 제압
3. 시체 남음

**Step 7.2: 명찰 루팅**
- 시체 근처 상호작용 가능
- UI 가이드:
  ```
  [E] - 명찰 루팅
  ⚠️ 주변에 다른 학생이 있으면 목격당합니다!
  ```

**분기 A: 안전하게 루팅 (주변 10타일 내 학생 없음)**
- [E] 입력 → 명찰 획득
- 시스템 메시지:
  ```
  [아이템 획득] 이름 명찰 (1개)

  명찰은 사망 시 대신 부서집니다 (부활권)
  ⚠️ 4개 이상 보유 시 선생님 출현
  ```

**분기 B: 목격당함 (주변에 학생 있음)**
- 시스템 메시지:
  ```
  [시스템] 목격당했다!
  ```
- 목격한 학생이 영구 추격 (손전등 무시)
- 플레이어 도주 필요

---

### Phase 8: 안전지대 도착 (8-10분)

**Step 8.1: 양호실 발견**
- 위치: 2층 복도 끝 (3층이 아닌 2층, 튜토리얼 단순화)
- UI 가이드:
  ```
  🏥 양호실
  안전지대입니다
  ```

**Step 8.2: 양호실 진입**
- 문 열기 → 양호실 내부
- 배경음악 변화 (긴장 → 안도)
- 학생 개체 진입 불가

**시스템 메시지**:
```
[시스템] 안전지대에 도착했습니다
```

**Step 8.3: 세이브 포인트**
- 침대 상호작용
- UI:
  ```
  ┌────────────────────────────┐
  │   세이브하시겠습니까?       │
  │                            │
  │   [예]    [아니오]         │
  └────────────────────────────┘
  ```
- [예] → 게임 저장

---

## ✅ 성공 기준

튜토리얼이 성공적이려면:
1. ✅ 플레이어가 10분 안에 핵심 메커니즘 이해
2. ✅ 즉사 횟수 2회 이하 (난이도 적절)
3. ✅ 첫 점멸까지 생존 (긴장감 유지)
4. ✅ UI 가이드가 명확하고 방해되지 않음
5. ✅ 자유 탐험 시작 시 자신감 획득
