## 1. 선생님 개체 설정

### 1.1 외형 (원작 기반)

#### 학생 개체와 완전히 다름
```
거대한 인간형 실루엣
여러 명의 선생님을 꾸깃꾸깃 뭉개고 이어붙여서
억지로 인간이라는 몰드에 다시 밀어 넣은 것 같은
시체골렘 형태

머리에 팔이 있고 어깨에 머리가 있고,
무릎에 어깨가 있는 살점 괴물

기묘하도록 단정한 발걸음 소리:
저벅저벅저벅저벅저벅저벅저벅
```

### 1.2 평상시 위치

- **5층에만 존재**
- **1-4층에는 전혀 없음**
- 명찰 4개 이상 소지 트리거가 발동되어야만 등장

### 1.3 핵심 특징

- **학생 개체와 구별됨**: 외형이 완전히 다름
- **손전등 무효화**: 손전등 빛으로 멈출 수 없음
- **명찰로 학생 인식**: 명찰 개수로 플레이어를 감지

## 2. 명찰 시스템과 선생님 출현

### 2.1 명찰 개수별 효과

| 명찰 개수 | 시스템 반응 | 선생님 상태 | 위험도 |
|:---:|:---|:---|:---:|
| **0개** | 없음 | 5층 대기 | 탈출 불가 |
| **1-2개** | 없음 | 5층 대기 | ✅ 안전 |
| **3개** | ⚠️ 경고 메시지만 | 5층 대기 | ⚠️ 주의 |
| **4개 이상** | 🚨 추격 시작 | **5층에서 등장 + 추격** | 🔴 매우 위험 |

### 2.2 명찰 3개 시스템 메시지

```
[시스템 경고]
선생님이 나를 감지할 수 있으니 주의하자.

⚠️ 주의: 명찰을 더 이상 모으지 마십시오.
```

**효과**:
- 메시지만 출력
- **선생님은 등장하지 않음**
- 실질적 위협 없음

### 2.3 명찰 4개 이상: 선생님 추격 시작

#### 트리거 발동 순간
```
[명찰 4개 획득]

[시스템 경고]
선생님이 내 위치를 감지했다!

💀 위험: 선생님이 5층에서 출발합니다!

[효과음]
저벅... 저벅... 저벅...
```

#### 추격 메커니즘
1. **선생님이 5층에서 등장**
2. **플레이어를 향해 이동 시작** (5층 → 4층 → 3층...)
3. **즉사 아님**: 도망칠 수 있음
4. **하지만 결국 따라잡힘**: 계속 추격당함

### 2.4 명찰을 버려도 추격은 계속됨

#### 시나리오
```
1. 명찰 4개 소지 → 선생님 추격 시작
2. 플레이어가 명찰 1개 버림 → 3개 소지
3. 선생님은 계속 추격함 (멈추지 않음)
```

#### 단, 감지 능력 감소
- 명찰 4개: 정확한 위치 파악
- 명찰 3개: 대략적 위치만 파악
- 명찰 2개: 감지 능력 크게 감소 (하지만 추격은 계속)
- 명찰 1개: 거의 감지 못함 (하지만 추격은 계속)

**결론**: 한 번 추격이 시작되면 명찰을 버려도 **따돌릴 수 없음**

## 3. 선생님 공격 및 대응 수단

### 3.1 선생님에게 데미지를 입히는 방법 (3가지만)

#### 1. 유리손포 + 유리구슬
```
데미지: 극소 (실질적 타격 거의 없음)
용도: 잠시 이동 저해, 시간 벌기
원작: "어깨에 맞은 탄환에 선생님이 괴로운 비명을 질렀다.
      하지만 실질적으로 그다지 타격은 없는 듯했다."
```

#### 2. 사인검 (해금 요원 전용, 해피엔딩 01 트리거 시)
```
데미지: 일시적 죽임
효과: 선생님이 일시적으로 소멸
한계: 잠시 후 부활함 (영구 소멸 불가)
용도: 시간 벌기
```

#### 3. 부적 완성 (해피엔딩 01)
```
데미지: 완전 소멸
효과: 선생님을 영구적으로 소멸시킴
조건: 유리 초롱 + 부적 복구 + 결계 발동
결과: 학교 전체 봉인 + 선생님 소멸
```

### 3.2 선생님 특성

- **손전등 무효화**: 손전등 빛으로 멈출 수 없음
- **도망 가능**: 플레이어보다 느리게 이동 (도망칠 수 있음)
- **하지만 지속 추격**: 결국 따라잡힘
- **층 이동**: 계단을 통해 모든 층 이동 가능

## 4. 6개 엔딩 상세

### 4.1 배드엔딩 01: 영원한 재진입 (Loop Ending)

#### 조건
```
명찰 0개 소지 상태에서
학생 개체에게 사망
```

#### 결말
- 선생님 등장하지 않음
- 명찰 없이 죽음
- 현실로 튕겨남
- 매달 그믐날마다 세광공고로 강제 소환되는 저주
- "졸업하지 못한 자는 영원히 학교를 벗어날 수 없다."

#### 보상
- **[추출 실패]** (None)

---

### 4.2 배드엔딩 02: 선생님의 저주 (Deletion Ending)

#### 조건
```
명찰 4개 이상 소지 +
선생님에게 사망 +
5층 강당 입성 전
```

#### 진행
1. 명찰 4개 이상 소지 → 선생님 등장 + 추격 시작
2. 플레이어가 도망치지만 결국 따라잡힘
3. 선생님에게 죽임당함
4. 5층 강당에 도달하지 못한 상태

#### 결말
- 욕심을 부린 대가
- 선생님에게 '학생'으로 인식되어 영혼이 학교에 귀속됨
- 현실 세계에서 김솔음의 존재(기록, 기억)가 **완전히 삭제**됨
- 게임 데이터(세이브) 삭제 연출

#### 보상
- **[추출 실패]** (None)

---

### 4.3 배드엔딩 03: 불완전한 탈출 (Death Ending)

#### 조건
```
명찰 1-3개 소지 상태에서
사망 (학생 개체 또는 기타 원인)
선생님 미등장 (4개 미만이므로)
```

#### 결말
- 김솔음은 사망(현실로 튕겨남)
- 하지만 소지한 명찰 덕분에 최소한의 꿈결 용액 추출 성공
- "죽었지만... 헛수고는 아니었어."

#### 보상
- **[꿈결 용액 B~C등급]** (Random Rare)

---

### 4.4 배드엔딩 04: 학교의 종말 (System Lockout Ending)

#### 조건
```
5층 강당 진입 +
졸업식 노래 40초 동안 생존 실패
```

#### 진행
1. 플레이어가 5층 강당에 입성
2. 졸업식 노래 시작 (40초)
3. 선생님의 공격을 피하지 못함
4. 또는 유리손포/구슬로 견제하지 못함
5. **졸업식 실패**

#### 결말 (원작 기반)
```
"본래 '배드엔딩'에서 저들은 모두
선생님의 수족이 되어 좀비 같은 꼴로 강당을 나와,
남아 있던 탐사자들도 다 죽여버리는데…."
```

- 학생들이 선생님의 수족이 되어 좀비화
- 강당에서 쏟아져 나옴
- 게임 강제 종료
- 화면 글리치 + 노이즈

```
[강제 종료]
"접속 불가합니다."
```

#### 보상
- **[접속 불가]** (System Lockout)
- 재시작 가능 (실제 6개월 차단 없음)

### 4.5 해피엔딩 02: 해방의 졸업식 (Normal Ending)

#### 조건
```
5층 강당 진입 +
졸업식 노래 40초 동안 생존 성공 +
부적 복구 실패 (뒤뜰 미진입)
```

#### 진행
1. 플레이어가 5층 강당에 입성
2. 졸업식 노래 시작 (40초)
3. 선생님의 공격을 유리손포/구슬로 견제하며 버팀
4. 40초 생존 성공
5. "졸업, 축하드립니다." 선언

#### 원작 장면
```
"졸업, 축하드립니다."

강당에 꽃보라 같은 홀로그램 픽셀이 가득 찬다.

괴성을 지르던 선생님이 녹아내리며 사라지고,
강당을 뒤덮고 있던 괴상한 이형체들도 함께 불타듯 사라진다.
학생들을 붙잡고 있던 의자도.

와아아아아!!

자유의 몸이 된 3학년 학생들이 터져나가듯 강당 밖으로 뛰쳐나갔다.
그들의 눈은 이제 눈물과 기쁨으로 범벅이다.

강당의 커튼이 걷히며, 창밖에서 노랗고 붉은 따스한 빛이
부드럽게 스며들어온다….
여명이었다.
```

#### 결말
- 졸업식 강행으로 학생들의 영혼 성불
- **선생님 일시적 소멸** (완전 소멸 아님)
- 학교는 소멸 (봉인되지 않음)
- 김솔음 혼자 생환
- 이결은 포함되지 않음 (부적 미완성)

#### 보상
- **[꿈결 용액 A등급]** (Epic)

### 4.6 해피엔딩 01: 도서관에 잠든 학교 (True Ending)

#### 조건
```
부적 완성 +
결계 발동 +
이결의 희생
```

#### 진행
1. 뒤뜰 진입 → 유리 초롱 + 찢어진 부적 획득
2. 새 부적 교체
3. 유리 초롱 복구 (흠집까지 부적의 일부)
4. 4층 도서실 소화전에 설치
5. 청동 요원의 축성
6. **이결이 유리 초롱을 잡음**
7. "교복 돌려받기" → 호도 요원의 오염 해제
8. 이결: "...또 봐."
9. **푸른 도깨비불 발동**

#### 원작 결계 발동 장면
```
도깨비불이 터지듯 타올랐다.

푸른 불꽃이 모든 것을 덮어 태운다.
불길한 게임 비트가 뒤틀린 듯한 소리가 사라지고.
살점에 뒤덮인 교실과 복도를 지우고.
학교를 전부 감싸며 불꽃의 그림자 속에 넣는다.
온통 시야가 하얗게 물든다.
```

#### 원작 엔딩 크레딧 (전문)
```
나는 눈을 떴다.
빛이 눈을 간지럽히고. 한낮의 교정이 내 시야에 펼쳐진다.
그리고 연주곡이 들린다.

피아노.

쇼팽의 녹턴.
천천히 흐르는 그 달콤하고 느린 피아노 소리에 맞추어.
1층 복도의 전경이 보인다.

학생들이 바닥에 누워 있다.
마치 한빛도서관을 통해 방문했던. 그믐날이 아닌 학교처럼.
하지만 결정적인 차이가 있다.

그들은 곤히 잠들어 있었다.

낮.
5월 초. 따끈한 햇살이 비치는 날에.
창문 밖에서는 조용한 녹음이 드리우며 복도를 비춘다.
안락한 자리가 마련되어 있다.

체육 시간에 쓰는 매트리스. 담요. 체육복과 키링 인형. 슬리퍼까지
온갖 푹신하고 좋은 것들이 복도에 누운 학생들의 자리마다 보인다.
친구와 손을 잡거나 편안한 체육복을 입고 잠든 학생들의 모습이 보인다.
가끔 이어폰을 낀 채로 잠들어 있기도 하다.
다 함께 즐기는 낮잠 시간.

…….

중간중간 비어 있던 자리도 채워져 있다.
다른 학교의 학생들이 그곳에 누워 있었던 것이다.
학교를 뒤덮은 살점에 먹혀 사지를 버둥대던 그들이.
학교와 일체화되었던 그들은 고등학교의 일부가 되었다.
하지만 이제 부드럽게 학생들의 빈틈으로 받아들여졌다.

'아.'

내 시야는 마치 누군가가 보여주듯이 위로 올라간다.
계단을 타고 위로. 학생들이 편안히 누워 낮잠을 자고 있는 풍경을
보여주며 가장 위층으로 올라간다.

5층.
선생님들과 장기 일부가 놓여 있던 강당 앞 복도에도 학생들이 누워 있다.
하지만 그 인상착의와 몇 가지 소지품에서 알아본다.
철뱃지.
요원들이다.
나는 강당의 문과 가까운 자리에서 햇빛을 받으며
낮잠을 자는 고영 요원의 얼굴을 알아본다.

시야가 올라가며….
굳게 닫힌. 멀쩡한 강당의 문을 보여준다.
그곳에는 아무도 없다.
다만 부서진 유리 초롱과. 가운데에 찢긴 자국이 남은 부적이
문손잡이에 걸려 햇살 속에서 반짝인다.
맡은 바 소명을 다한 듯이.

```

#### 결말
- **선생님 완전 소멸**

## 5. 해피엔딩 01 게임화 상세

### 5.1 이결 희생 이벤트 (핵심)

#### 원작 대화
```
[결계 완성 직전]

이결: "교복은 지금 도로 돌려받았다고 칠게.
      애초에 거짓말이었으니까."

솔음: "...!"

이결: "넌 이 학교 학생 아니야."
```
