## 1. 플레이어 수치

### 1.1 이동 속도 (개선)

|:---:|:---:|:---:|:---:|
| **걷기 (기본)** | 6.0 | 100% (기준) | 중간 |
| **달리기 (Shift)** | 9.0 | 150% | 높음 (어그로 대폭 증가) |
| **앉기/숨기 (Ctrl)** | 3.0 | 50% | 낮음 |

**변경 이유**: 회복 아이템 정당화 + 긴장감 유지 (1회 실수는 용서, 2회 실수는 사망)

| 항목 | 수치 | 비고 |
|:---:|:---:|:---|
| **공격력 (기본 주먹)** | 35 | 학생 개체 3-hit kill |
| **공격 쿨타임** | 0.5초 | 공격 간 대기 시간 |
| **공격 범위** | 1타일 | 근접 거리 |

- 5-hit → 3-hit으로 감소 (전투 속도 개선)
- 은신 처치 추가 (전략적 선택지 제공)
- 전투 vs 은신의 risk/reward 명확화

## 2. 적 개체 수치

### 2.1 일반 학생 개체 (메커니즘 명확화)
| 항목 | 수치 | 비고 |
|:---:|:---:|:---|
| **HP** | 100 | 3회 공격 또는 은신 1회로 처치 |
| **배회 속도** | 2.0 (타일/초) | 플레이어 걷기의 33% |
| **추격 속도** | 6.0 (타일/초) | 플레이어 걷기의 100% (같음) |
| **추격 발동 조건** | **시선 가이드라인 정면 접촉** (추격 개시) | 손전등 빛과 무관 |
| **추격 해제 조건** | 시야 벗어남 + 15타일 이상 | 문 격리, 은신처 진입 |
| **정지 조건** | **손전등 빛 범위 내** 진입 | 추격 중이어도 빛 안에서 정지 |
| **공격력** | 50 | 접촉 시 플레이어 HP 50 감소 |
| **어그로 범위 (소음)** | 반경 20타일 | 소음 발생 시 몰려옴 |
| **어그로 범위 (밝은 빛)** | 반경 10타일 | 손전등 '강' 모드 사용 시 |

**⭐ 핵심 메커니즘 명확화**:
1. **시선 가이드라인 (정면/뒤 구분)**:
   - 마우스 방향 점선이 학생 개체의 **정면(시야각 내)**에 닿으면 → **추격 개시** (영구적)
   - 학생 개체의 **뒤**에 시선이 닿으면 → **효과 없음** (추격/정지 발동 안 됨)
   - 뒤에서 시선으로 보면 아무 일 없이 지나칠 수 있음
2. **손전등 빛 (방향 무관)**:
   - 손전등 범위 내 학생 개체는 **방향 무관**하게 모두 정지
   - 추격 중인 학생을 정지시킴 (빛 끄면 다시 추격)
3. **추격 해제**: 시야 벗어나거나 문/은신처로 격리 시 추격 중단
4. **재추격**: 추격이 해제된 학생도 다시 정면 시선에 들어오면 재개시

### 2.2 층별 배치 밀도 (대폭 변경)
| 층 | 학생 개체 수 | 배치 타입 | 비고 |
|:---:|:---:|:---:|:---|
| **1층** | 4-6명 | 드문드문 배치 | 깜짝 놀람 요소 |
| **2층** | 15-20명 | 높은 밀도 (순찰형) | 주요 배회 구역, 시작 지점 |
| **3층** | 15-20명 | 높은 밀도 (순찰형) | 주요 배회 구역 |
| **4층** | **60명 (30명 구속 + 30명 추가)** | **고정형 + 배회** | 모든 복도에 배치 ⚠️ |
| **5층** | 선생님 (보스) + 300명 (의자 일체화) | 보스 | 최종 보스전 |

**4층 상세 배치**:
- **복도 4개 모두**에 학생 개체 양옆으로 줄 서 있음
- 구속된 팔 형태 (발이 묶임, 이동 불가)
- 창고 문 앞: 5-6명 밀집 배치
- **통과 조건**: 학생 처치 or 전학생 모드 착용
- **층 격리**: 4층 학생은 4층을 벗어날 수 없음 (졸업식 트리거 전까지)

**층 격리 규칙 (중요)**:
- 각 층의 학생 개체는 해당 층을 절대 벗어날 수 없음
- 1층 개체 → 1층만 / 2층 개체 → 2층만 / 3층 개체 → 3층만 / 4층 개체 → 4층만
- **예외**: 졸업식 트리거 시 4층 개체 전부 사라지고 5층 강당으로 이동

### 2.3 특수 적 - 목격 상태 학생 (삭제됨)
**변경**: 명찰 루팅 목격 시 영구 추격 메커니즘 삭제
**이유**: 시선 추격 시스템만으로 충분한 긴장감 제공

### 2.4 선생님 (보스) - 최종 확정

#### 기본 정보
| 항목 | 수치 | 비고 |
|:---:|:---:|:---|

| **이동 속도** | 5.0 (타일/초) | 플레이어보다 느림 (도망 가능) |
| **공격력** | 100 (즉사) | 접촉 시 사망 |
| **손전등 효과** | 무효 | 손전등으로 멈출 수 없음 |

#### 추격 메커니즘
- **5층에서 출발**: 선생님은 평상시 5층에만 존재
- **즉사 아님**: 도망칠 수 있음 (플레이어보다 느림)

  - 3개: 대략적 위치만 파악
  - 2개: 감지 능력 크게 감소
  - 1개: 거의 감지 못함

### 3.3 시선 시스템 (핵심 메커니즘)
| 항목 | 수치 | 비고 |
|:---:|:---:|:---|
| **시선 가이드라인** | 점선 형태, 마우스 방향 | 실시간 표시 |
| **시선 최대 거리** | 25타일 | 손전등보다 길지만 무한 아님 |
| **벽 관통** | 불가능 | 벽/가구에 막힘 |
| **유리/창문 관통** | 가능 | 투명 장애물 통과 |
| **추격 개시 조건 (정면)** | 가이드라인이 학생 **정면(시야각 내)**에 닿을 때 | 눈이 마주침 |
| **추격 개시 조건 (뒤)** | 가이드라인이 학생 **뒤**에 닿으면 **효과 없음** | 추격/정지 발동 안 됨 |
| **추격 개시 메시지 (첫 3회)** | "시선이 마주쳤다. 이제 저 학생 개체는 죽을 때까지 나를 쫓아올 것이다." | |
| **추격 개시 이펙트 (4회부터)** | 비주얼 효과 (눈 아이콘 깜빡임) | 시스템 메시지 없음 |
| **정지 조건 (손전등)** | 손전등 빛 범위 내 (방향 무관) | 모든 학생 개체 정지 |
| **정지 조건 (시선)** | 불가능 | 시선만으로는 정지 안 됨 |

**중요**:
- **시선 = 추격 트리거 (정면만)** / **손전등 = 정지 트리거 (방향 무관)**

## 8. 밸런싱 참고 공식

### 8.1 추격 성공률
```
추격 성공률 = (적 속도 - 플레이어 속도) / 평균 도주 거리

- 일반 학생: (6.0 - 6.0) / 15타일 = 0% (같은 속도, 코너링으로 도주)
- 묵념 중: (8.0 - 6.0) / 15타일 = 13% (도주 어려움)
```

### 8.2 점멸 생존 시간
```
예고 시간(5초) + 평균 은신 이동 시간(3초) = 8초
점멸 주기(300초) - 생존 준비 시간(8초) = 292초 (안전 시간)

4층: 점멸 주기(180초) - 8초 = 172초 (안전 시간 짧음)
```

### 8.3 손전등 배터리 관리
```
비상 손전등 (10분):
- 약 모드: 10분 지속
- 중 모드: 6.6분 지속
- 강 모드: 5분 지속

공업용 손전등 (30분):
- 약 모드: 30분 지속
- 중 모드: 20분 지속
- 강 모드: 15분 지속
```
