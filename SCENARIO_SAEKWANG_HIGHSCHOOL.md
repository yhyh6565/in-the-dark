## Scene 0-2: 깨어남 (게임 시작)

**Location**: 2층 1학년 5반 교실
**Time**: 심야 (점등 전)
**Characters**: 김솔음(플레이어), 혼란에 빠진 시민들, 정지한 학생 개체 1명
**Duration**: 2분
**Emotional Tone**: 혼란, 불안, 긴장

### 환경 상태

- **조명**: 어둡지만 창문을 통해 들어오는 희미한 달빛
- **교실 배치**: 학생 책상 8x4 배열, 교탁, 칠판, 뒷문/앞문
- **NPC 배치**:
  - 시민 8명 (책상에 엎드려 있다가 일어남)
  - 학생 개체 1명 (뒷문 근처, 완전 정지)

### 시퀀스

**1. 깨어남 (0~10초)**

```
[화면이 천천히 밝아짐]
[1인칭 시점에서 책상 위를 내려다보는 장면]

[시스템 메시지]
접속 완료.

[시스템 메시지]
현재 신분: 1학년 5반 김솔음

[플레이어 컨트롤 활성화]
[카메라가 탑다운 쿼터뷰로 전환]
```

**2. 혼란의 시작 (10~30초)**

```
[교실 내 다른 NPC들도 깨어남]

시민 A: "어? 어디야 여기?"
시민 B: "저기요. 이게 무슨...?"

[웅성거림]
[일부는 일어나서 주변을 둘러봄]
[일부는 여전히 혼란스러워하며 앉아있음]

김솔음: (내면)
'여기가... 세광공업고등학교.'

김솔음: (내면)
'접속에 성공했다.'

[플레이어가 WASD키로 이동 가능]
[튜토리얼 UI: "WASD - 이동"]
```

**3. 학생 개체 발견 (30~50초)**

```
[시민들이 뒷문 쪽의 정지한 학생을 발견]

시민 C: "저기... 저게 뭐야?"
시민 D: (다가가며) "인형인가?"

[정지한 학생 개체]
[얼굴 이목구비 없음, 교복 착용, 미동도 없음]

시민 E: "으악! 깜짝이야!"
시민 F: (불안하게) "장난... 아니지?"

김솔음: (내면)
'시작됐다.'

김솔음: (내면)
'복잡한 감정은 나중에. 지금은... 살아야 한다.'

[손을 내려다보는 연출]
[플레이어 캐릭터의 손 클로즈업]

김솔음: (내면)
'다행히 인간의 모습이다. 세광특별시 내부라 그런가.'

[자신의 교복 확인]
[타 고등학교 교복 - 남색 자켓, 회색 바지]

김솔음: (내면)
'예전 고등학교 교복... 학교 이름만 지워져 있네.'
```

**4. 첫 번째 방송 예고 (50~60초)**

```
[스피커에서 정적 소리]
[찌지직...]

[전체 공지 - 화면 중앙 상단]
"딩동댕동, 세광공업고등학교 학생 여러분!"

[짧은 정적]

'곧... 온다.'

## Scene 1-2: 복도 탈출

**Location**: 2층 복도
**Duration**: 3분
**Emotional Tone**: 긴장, 공포
**Gameplay Focus**: 시선 메커니즘 학습, 손전등 사용

### 환경 상태

- **조명**: 깜박이는 형광등
- **학생 개체**: 5명 배회 (느린 속도)
- **소음**: 복도 끝에서 비명 소리

### 시퀀스

**1. 복도 진입 (0~30초)**

```
[뒷문을 열고 복도로 나옴]
[탑다운 쿼터뷰]

[복도 전경]
- 길이 40m의 일직선 복도
- 좌우로 교실 문들
- 학생 개체 5명이 느리게 배회 중

[시스템 메시지]
근처에 무언가 느껴진다

김솔음: (내면)
'복도에 5명... 시야 확보가 중요해.'

[튜토리얼 UI]
학생 개체는 평소 매우 느리게 움직입니다 (플레이어의 0.5배)
시선이 닿으면 빠르게 추격합니다 (플레이어의 1.5배)
```

**2. 손전등 사용 학습 (30~60초)**

```
[학생 개체 A가 플레이어 방향으로 느리게 이동]

김솔음: (내면)
'앞에... 뭔가 있어.'

[플레이어가 마우스로 방향 조준]
[시선 가이드라인 표시 - 희미한 점선]

[시선 가이드라인이 학생 개체 A에 닿음]

[학생 개체 A - 추격 모드 진입]
[속도 급증]
[빠르게 접근]

[튜토리얼 UI - 긴급]
! 위험: 추격 당하는 중
[F] 키로 손전등을 켜세요!

[플레이어가 [F] 키 입력]

[손전등 ON]
[원뿔 형태의 빛 - 45도 각도, 10m 거리]

[학생 개체 A - 빛에 닿음]
[즉시 정지]
[움직이지 않음]

[배터리 UI 표시]
배터리: 100% → 95% (5초 경과)

김솔음: (내면)
'통한다...! 하지만 배터리가 계속 줄어들어.'

[튜토리얼 UI]
손전등 빛을 비추면 학생 개체가 정지합니다
배터리가 한정되어 있으니 아껴 사용하세요
```

**3. 우회 이동 (60~120초)**

```
[학생 개체 A를 손전등으로 정지시킨 상태]

[튜토리얼 UI]
정지한 학생 개체를 응시한 채로 옆으로 우회하세요
(WASD 이동 + 마우스 조준 유지)

[플레이어가 천천히 옆으로 이동]
[학생 개체 A와 충분한 거리 확보]

김솔음: (내면)
'이제 안전한 거리야. 손전등을 꺼야겠어.'

[플레이어가 [F] 키로 손전등 OFF]

[학생 개체 A - 다시 느린 배회 시작]

[배터리 UI]
배터리: 87%
```

**4. 추가 학생 개체 조우 (120~180초)**

```
[복도 중간 지점]
[학생 개체 B, C가 복도를 막고 있음]

김솔음: (내면)
'두 명이나... 손전등을 아껴야 해.'

[선택지]
> 손전등 없이 시야 밖으로 우회한다 (숙련자용)
  손전등을 켜고 안전하게 통과한다 (초보자용)
  소리폭탄을 사용한다 (아이템 소모)

[플레이어 선택에 따라 진행]

[손전등 없이 우회 선택 시]
