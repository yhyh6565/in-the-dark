## 1. 건물 전체 구조

### 1.1 기본 정보
- **구조**: ㅁ자형 5층 건물
- **맵 크기**: 약 50x50 타일 (각 층)
- **중정**: 건물 중앙 빈 공간 (약 15x15 타일)
- **계단**: 3개 (남서, 남동, 북중앙)

### 1.2 층별 개요

| 층 | 테마 | 주요 공간 | 색상 필터 |
|:---:|:---:|:---|:---|
| **1층** | 공용 공간 | 식당, 도서실, 행정실 | 회색 (어두움) |
| **2층** | 1학년 교실 | 1-1 ~ 1-8반, 양호실 | 연한 파랑 |
| **3층** | 2학년 교실 | 2-1 ~ 2-7반, 양호실 (안전지대) | 연한 초록 |
| **4층** | 3학년 교실 | 3-1 ~ 3-5반, 특수실, 창고 | 보라색 (완전 암흑) |
| **5층** | 강당 | 강당 (졸업식장) | 붉은색 (공포) |

### 1.3 계단 연결

```
계단1 (남서쪽):  1층 ↔ 2층 ↔ 3층 ↔ 4층 ↔ 5층
계단2 (남동쪽):  1층 ↔ 2층 ↔ 3층 ↔ 4층 (5층 차단)
계단3 (북중앙):  1층 ↔ 2층 ↔ 3층 ↔ 4층 ↔ 5층
```

## 2. 1층 레이아웃

### 2.1 전체 맵 (50x50 타일)

```
북쪽
↑
│     ┌───────────────────────────────────────────────┐
│     │ 화장실(남) 화장실(여)  계단3   화장실(남) 화장실(여) │
│     │    W1        W2        S3        W3        W4    │
│     ├─────────────────────────────────────────────────┤
│     │                                                   │
│     │   도서실         │ 중정(中庭) │      음악실        │
│     │   Library      │  Court    │    Music Room     │
│     │                 │           │                   │
서쪽──│                 │           │                   │──동쪽
│     │   상담실         │           │      식당          │
│     │   Counseling   │           │   Cafeteria       │
│     │                 │           │                   │
│     │                 │           │      행정실        │
│     │                 │           │      Admin        │
│     ├─────────────────────────────────────────────────┤
│     │  계단1    본관 현관(봉쇄)      계단2              │
│     │   S1      Main Entrance       S2                │
│     └───────────────────────────────────────────────────┘
↓
남쪽
```

### 2.2 상세 치수

**중정 (Court)**:
- 크기: 15x15 타일
- 위치: 중앙
- 타입: 통행 불가 (내벽 + 외벽)
- 창문: 내벽에 창문 타일 (4방향)

**도서실 (Library)**:
- 크기: 10x12 타일
- 위치: 서쪽 상단
- 오브젝트: 책장 6개, 책상 4개, 의자 8개
- 문: 복도 쪽 1개

**식당 (Cafeteria)**:
- 크기: 12x15 타일
- 위치: 동쪽
- 오브젝트: 긴 테이블 6개, 의자 24개, 자판기 2개
- 문: 복도 쪽 2개

**화장실 (Bathroom) × 4**:
- 크기: 각 6x8 타일
- 위치: 북쪽 좌우 대칭
- 오브젝트: 칸막이 3개, 세면대 2개
- 문: 복도 쪽 1개

**계단**:
- 크기: 각 4x6 타일
- 계단1 (S1): 남서쪽
- 계단2 (S2): 남동쪽
- 계단3 (S3): 북중앙

### 2.3 학생 개체 배치

| 위치 | 초기 배치 수 | 순찰 경로 |
|:---:|:---:|:---|
| **복도** | 2명 | 중정 주변 순찰 |
| **도서실** | 1명 | 책장 사이 배회 |
| **식당** | 1명 | 테이블 사이 배회 |
| **화장실** | 0명 | - |
| **총계** | **4명** (낮은 밀도) | |

### 2.4 아이템 배치

| 아이템 | 위치 | 개수 |
|:---:|:---|:---:|
| **배터리** | 화장실 세면대 옆, 복도 바닥, 특수실 | 10개 |
| **동전** | 자판기 밑, 복도 바닥 | 8개 |
| **손전등** | 특수실(식당, 도서실, 행정실 등)별 | 3개씩 |

│     ├─────────────────────────────────────────────────┤
│     │                                                   │
│     │  1-1  1-2  1-3  │ 중정 │  1-4  1-5  1-6        │
│     │                 │      │                        │
서쪽──│                 │      │                        │──동쪽
│     │  1학년          │      │          1-7  1-8      │
│     │  교무실         │      │                        │
│     │                 │      │      양호실            │
│     │                 │      │      (Infirmary)       │

### 3.2 교실 상세 (표준)

**1-5반 (시작 지점)**:
- 크기: 9x12 타일
- 레이아웃:
  문(위쪽)
  │
  ├─────────────────┐
  │ 칠판 (Blackboard)│
  │ [교무 책상]      │
  ├─────────────────┤
  │ [D][D][D][D][D] │  D = 책상(Desk)
  │                 │
  │ [D][D][D][🌟D][D] │  🌟 = 김솔음 시작 위치
  │                 │
  │ [D][D][D][D][D] │
  │                 │
  │ [L][L] [거치대] │  L = 사물함(Locker)
  └─────────────────┘
  ```
- 총 책상: 15개
- 사물함: 후면 2개
- 손전등 거치대: 후면 1개 (뒷문 옆)
- 칠판: 전면 1개 (3타일 × 2타일)
- 교무 책상: 전면 1개

**교실 공통 사양** (1-1 ~ 1-8):
- 크기: 9x12 타일
- 책상: 15개 (5열 × 3행)
- 사물함: 2개
- 칠판: 1개
- 문: 복도 쪽 (위쪽)

### 3.3 1학년 양호실 (Infirmary)

- 크기: 10x10 타일
- 레이아웃:
  ```
  문(왼쪽 복도)
  │
  ├──────────────┐
  │ [침대] [침대]│
  │ [침대] [침대]│
  │              │
  │ [서랍장]     │
  │ [약품 진열장]│
  │ [책상]       │
  └──────────────┘
  ```
- 침대: 4개 (커튼 포함)
- 서랍장: 1개 (세광공고 교복 보관, 3층 양호실)
- 약품 진열장: 1개

### 3.4 학생 개체 배치 (High Density)

| 위치 | 초기 배치 수 | 순찰 경로 |
|:---:|:---:|:---|
| **복도** | 8명 | 전체 복도 순찰 |
| **1-1 ~ 1-4반** | 각 2명 | 교실 내부 배회 |
| **1-6 ~ 1-8반** | 각 2명 | 교실 내부 배회 |
| **1-5반** | 0명 | 시작 지점 (안전) |
| **양호실** | 0명 | - |
| **총계** | **20명** (높은 밀도) | |

### 3.5 아이템 배치

| 아이템 | 위치 | 개수 |
|:---:|:---|:---:|

│     │                                                   │
│     │  3-1  3-2  3-3  │ 중정 │  3-4  3-5             │
│     │  🧟🧟🧟         │      │        🧟🧟🧟          │
│     │  [구속된 팔]    │      │    [구속된 팔]         │
서쪽──│                 │      │                        │──동쪽
│     │  3학년          │      │          도서실        │
│     │  교무실         │      │                        │
│     │                 │      │      음악실            │
│     │                 │      │                        │

🧟 = 구속된 팔 (고정형 적)
```

### 5.2 구속된 팔 배치 (The Bound Arms)

**3-1 ~ 3-4반 앞 복도** (서쪽):
```
┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐
│ 3-1 │ │ 3-2 │ │ 3-3 │ │ 3-4 │
└─────┘ └─────┘ └─────┘ └─────┘
🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟  ← 학생 고정 (이동 불가)
       [복도 통로]
🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟🧟  ← 학생 고정
```

**공격 패턴**:
- 플레이어가 지나갈 때 랜덤하게 팔을 휘두름
- 손전등으로 정지 가능
- 앉아서 지나가면 회피 가능 (느림, 위험)

### 5.3 창고 (Storage)

**위치**: 동쪽 복도 끝
**크기**: 8x10 타일
**레이아웃**:
```
문(복도)
│
├──────────────┐
│ [상자][상자] │
│ [상자][상자] │
│              │
│ [사다리]     │
│     ↑        │
│  환풍구 접근  │
│              │
│ 🎗️ 명찰 장식 │  🎗️ = 필수 아이템
└──────────────┘
```

**진입 조건**:
- 창고 앞 학생 무리에게 [종이와 펜]으로 쪽지 남기기
- 열쇠 획득 후 문 개방

### 5.4 학생 개체 배치

| 위치 | 초기 배치 수 | 타입 |
|:---:|:---:|:---|
| **복도 (구속된 팔)** | 30명 | 고정형 (이동 불가, 팔 공격) |
| **교실 내부 (추가)** | 20명 | 일반형 (교실 내 배회) |
| **창고 앞** | 5명 | 일반형 (배회) |
| **음악실** | 2명 | 일반형 |
| **도서실** | 3명 | 일반형 (추가 1명) |
| **총계** | **60명** | **4층 고밀도 구간** |

- **층 격리**: 4층 학생들은 절대 3층 이하로 내려갈 수 없음

### 9.1 문 (Door) 상호작용

#### 9.1.1 자동 개폐 문 (일반 교실, 화장실, 특수실)

**트리거 조건**:
- 플레이어 히트박스 중심이 문 타일로부터 **2타일 이내** 접근 시 자동 개방
- 2타일 이상 멀어지면 자동 폐쇄 (0.5초 딜레이)

**애니메이션**:
- 개방 애니메이션: **0.3초** (3프레임)
- 폐쇄 애니메이션: **0.3초** (3프레임)
- 문이 열리는 동안 충돌 박스 비활성화

**시스템 흐름**:
```
[플레이어 접근] (2타일 이내)
  ↓
[문 감지] (Area2D body_entered)
  ↓
[개방 애니메이션 재생] (0.3초)
  ↓
[충돌 박스 비활성화]
  ↓
[플레이어 통과 가능]
  ↓
[플레이어 멀어짐] (2타일 이상)
  ↓
[0.5초 대기] (안전 딜레이)
  ↓
[폐쇄 애니메이션 재생] (0.3초)
  ↓
[충돌 박스 활성화]
```

**Godot 구현 참고**:
- `AnimatedSprite2D`: 문 애니메이션 (open/close/idle)
- `Area2D`: 2타일 반경 감지 영역 (Radius: 32px)
- `StaticBody2D`: 문 충돌 박스 (닫혔을 때만 활성화)
- `CollisionShape2D`: 2타일 × 1타일 (32x16px)

- [ ] 자동 개폐 문 (2타일 감지, 0.3초 애니메이션)
- [ ] 잠긴 문 (창고 열쇠 사용 시스템)
- [ ] 봉쇄 문 (1층 본관 현관, 조사 가능)
- [ ] 강당 문 (명찰 장식 착용 조건)

#### 계단 시스템
- [ ] 계단 진입 감지 (히트박스 50% 겹침)
- [ ] 층 전환 애니메이션 (페이드 0.5초 × 2)
- [ ] 계단별 연결 좌표 설정
- [ ] 계단2 차단 처리 (4층 → 5층)
- [ ] 학생 개체 계단 이동 딜레이 (1.0초)

#### 창문 시스템
- [ ] 3층 양호실 창문 (뒤뜰 진입)
- [ ] 전학생 모드 조건 체크
- [ ] 뒤뜰 씬 전환 (페이드 흰색)
