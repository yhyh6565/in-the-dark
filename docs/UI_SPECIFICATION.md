## 1. 기준 해상도 및 스케일

### 1.1 기본 설정

| 항목 | 값 | 비고 |
|:---|:---:|:---|
| **기준 해상도** | 1920x1080px | Full HD |
| **최소 해상도** | 1280x720px | HD |
| **UI 스케일링** | Pixel Perfect | 정수배 확대만 허용 |
| **폰트** | 8비트 비트맵 폰트 | 픽셀 완벽 정렬 |
| **UI 레이어** | 최상위 (Z-index 100) | 게임 화면 위 |

### 1.2 색상 팔레트

8비트 스타일 제한 색상:

```
배경: #1A1A1A (거의 검은색)
텍스트: #EEEEEE (흰색)
강조: #FF6B6B (빨강)
경고: #FFD93D (노랑)
안전: #6BCF7F (초록)
UI 프레임: #4A4A4A (회색)
```

## 2. HUD (Head-Up Display)

> **출처**: SYSTEM_BALANCING.md v2.1 Section 5

### 2.1 전체 레이아웃

```
화면 1920x1080px 기준

┌────────────────────────────────────────────────────┐
│ HP: ■■■■■■■■■■ 100/100          점멸: ■■■■■ 180s │ ← 상단
│ 배터리: ▓▓▓▓▓▓▓░░░ 450/600        🔦 중           │
│                                                    │
│                                                    │
│                                                    │
│                (게임 화면)                          │
│                                                    │
│                                                    │
│                                                    │
│                                                    │
│                                    [F] 문 열기     │ ← 프롬프트
│                                                    │
│                                                    │
│ 층: 2층                            💾 저장 중...   │ ← 하단
└────────────────────────────────────────────────────┘
```

### 2.2 HP 바

**위치**: 좌측 상단 (20px, 20px)
**크기**: 200px × 20px
**구성**:
```
┌────────────────────┐
│ HP: ■■■■■■■■■■    │ ← 현재 HP (초록/노랑/빨강)
│     100/100        │ ← 숫자 텍스트
└────────────────────┘
```

**색상 변화**:
- 80-100 HP: 초록 (#6BCF7F)
- 40-79 HP: 노랑 (#FFD93D)
- 0-39 HP: 빨강 (#FF6B6B)

**애니메이션**:
- 데미지 받을 시: 빨간색 깜빡임 (0.3초)
- HP 감소: 부드러운 트윈 (0.5초)

### 2.3 점멸 타이머 바

**위치**: 우측 상단 (1700px, 20px)
**크기**: 200px × 20px

```
┌────────────────────┐
│ 점멸: ■■■■■ 180s   │ ← 타이머 바
└────────────────────┘
```

**색상 변화** (2층 기준 300초):
- 0-235초: 초록 (#6BCF7F)
- 235-295초: 노랑 (#FFD93D)
- 295-300초: 빨강 (#FF6B6B) + 깜빡임 (0.5초 주기)

**층별 표시**:
- 1층: "7:00", 2층: "5:00", 3층: "4:00", 4층: "3:00"
- 5층: 표시 안 함 (점멸 없음)

### 2.4 배터리 게이지

**위치**: 좌측 상단 두 번째 줄 (20px, 50px)
**크기**: 200px × 20px

```
┌────────────────────┐
│ 배터리: ▓▓▓▓▓░░░░░│ ← 잔량 바
│         450/600    │ ← 초 단위
└────────────────────┘
```

**색상**:
- 80-100%: 초록
- 40-79%: 노랑
- 0-39%: 빨강 + 깜빡임

**표시 조건**: 손전등 켜져 있을 때만 표시

### 2.5 손전등 밝기 아이콘

**위치**: 우측 상단 두 번째 줄 (1700px, 50px)
**크기**: 64px × 32px

```
[F] 🔦 약   ← 약
[F] 🔦 중   ← 중
[F] 🔦 강   ← 강
[F] 🔦 꺼짐 ← 꺼짐
```

**F키 토글 순서**: 약 → 중 → 강 → 꺼짐 → 약

### 2.6 층 표시

**위치**: 좌측 하단 (20px, 1040px)
**크기**: 100px × 20px

```
층: 2층
```

### 2.7 자동 저장 아이콘

**위치**: 우측 하단 (1800px, 1040px)
**크기**: 100px × 20px

```
💾 저장 중...
```

**애니메이션**: 회전 (0.5초 주기)
**지속 시간**: 1초 (성공), 3초 (실패)

## 3. 인벤토리 UI

> **출처**: CHARACTERS_AND_ITEMS.md v1.4 Section 0

### 3.1 인벤토리 창

**트리거**: Tab 키
**위치**: 화면 중앙
**크기**: 800px × 600px

```
┌──────────────────────────────────────┐
│           인벤토리 (10/10)            │
├──────────────────────────────────────┤
│ [0] 🔦        [5] (빈 슬롯)           │
│ 비상 손전등   450/600s               │
│                                      │
│ [1] 🔋×12     [6] (빈 슬롯)           │
│ 배터리                               │
│                                      │
│ [2] 🏷️         [7] (빈 슬롯)           │
│ 명찰: 최민준                          │
│                                      │
│ [3] 🏷️         [8] (빈 슬롯)           │
│ 명찰: 박서연                          │
│                                      │
│ [4] 🪙×5      [9] (빈 슬롯)           │
│ 동전                                 │
├──────────────────────────────────────┤
│ [선택한 아이템]                       │
│ 비상 손전등                           │
│ - 수명: 450/600초                    │
│ - 밝기: 약/중/강 조절 가능            │
│                                      │
│ [사용]  [버리기]  [닫기]              │
└──────────────────────────────────────┘
```

### 3.2 슬롯 배치

**그리드**: 2열 × 5행
**슬롯 크기**: 64px × 64px
**간격**: 16px

```
[0] [5]
[1] [6]
[2] [7]
[3] [8]
[4] [9]
```

### 3.3 아이템 정보 패널

**위치**: 인벤토리 창 하단
**크기**: 760px × 150px

**표시 내용**:
- 아이템 이름
- 수량 (중첩 가능 아이템)
- 내구도 (손전등)
- 설명 텍스트

### 3.4 버튼

**크기**: 각 100px × 32px
**배치**: 하단 중앙 정렬

- **[사용]**: 회복약 사용, 특수 아이템 장착
- **[버리기]**: 아이템 드롭 (확인 UI 표시)
- **[닫기]**: Tab 키 또는 ESC로도 가능

## 4. 상호작용 프롬프트

### 4.1 문 상호작용

**트리거**: 문 2타일 이내 접근
**위치**: 화면 우측 하단 (1600px, 900px)
**크기**: 200px × 40px

```
┌────────────┐
│ [F] 문 열기 │
└────────────┘
```

**자동 개폐 문**:
- 플레이어 접근 시 자동 개방 (프롬프트 없음)

**잠긴 문**:
```
┌──────────────────┐
│ [F] 문이 잠겼다   │
│   (창고 열쇠 필요)│
└──────────────────┘
```

### 4.2 아이템 습득 프롬프트

**트리거**: 아이템 1타일 이내
**위치**: 아이템 위 (상대 좌표)

```
     ↓
┌────────┐
│ [F] 줍기│
└────────┘
   [아이템]
```

### 4.3 NPC 대화 프롬프트

**트리거**: NPC 2타일 이내

```
┌─────────────┐
│ [F] 대화하기 │
└─────────────┘
```

### 4.4 세이브 포인트 프롬프트

**트리거**: 양호실 책상 앞

```
┌───────────────┐
│ [E] 저장하기   │
└───────────────┘
```

## 6. 게임 오버 UI

> **출처**: SAVE_SYSTEM.md v1.0 Section 5

### 6.1 사망 시퀀스

**1단계 (0-2초)**: 사망 애니메이션
- 캐릭터 쓰러짐
- 화면 shake (진폭 10px, 0.5초)

**2단계 (2-3초)**: 페이드 아웃
- 빨간색 비네팅 효과
- 검은색으로 페이드 (1초)

**3단계 (3-6초)**: 대기
- 검은 화면 유지 (3초)

**4단계 (6초~)**: 게임 오버 UI 표시

### 6.2 게임 오버 화면

**위치**: 화면 중앙
**크기**: 800px × 600px

```
┌──────────────────────────────────────┐
│                                      │
│           GAME OVER                  │
│                                      │
│     사망 원인: 학생 개체 공격         │
│                                      │
│     생존 시간: 0:45:23               │
│     명찰 수집: 2개                    │
│                                      │
│                                      │
│   ▶ 처음부터 시작                     │
│     마지막 저장 지점에서 로드          │
│     메인 메뉴로                       │
│                                      │
└──────────────────────────────────────┘
```

**조작**:
- ↑↓ 키: 선택지 이동
- Enter: 선택 확인
- ESC: 메인 메뉴로
