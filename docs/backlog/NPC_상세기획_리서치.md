## 1. 학생 개체 (Student Entities)

> **중요**: 본 섹션의 내용은 기존 게임 기획 문서(GAME_DESIGN_SAEKWANG_HIGHSCHOOL.md, SYSTEM_BALANCING.md)의 내용과 상충할 경우 **기존 기획안 문서가 우선**됩니다. 이 리서치는 참고 자료이며, 명시적 승인 없이 기존 설계를 대체하지 않습니다.

### 1.1 핵심 행동 패턴

**주요 메커니즘: Weeping Angel 시스템**
- **원작 인용**: "세광공업고등학교의 학생은 누군가가 쳐다보는 동안에는 움직이지 않는다. 그러나 시선이 없어지는 순간..."
- **기존 기획안 참조**: SYSTEM_BALANCING.md v2.0 Section 3.3 (시선 시스템)

**이동 규칙**:
1. **관찰 상태 (시선 있음)**:
   - 완전히 정지, 밀랍인형이나 마네킹처럼
   - 안전하게 관찰 가능
   - 시선을 유지하며 접근 가능

2. **비관찰 상태 (시선 없음)**:
   - 초자연적인 속도로 이동
   - 가장 가까운 목표 추격
   - 점멸/암전 시 위치 재배치

3. **소리 감지**:
   - **원작 인용**: "학생들도, 소리를 들은 것이다."
   - 큰 소리에 끌림
   - 전략적으로 유인 가능

**외형**:
- 얼굴: 무표정 (얼굴이 없음)
- 의복: 온전한 세광공고 교복
- **명찰**: 가슴에 부착 - 핵심 게임플레이 아이템
- 신체 상태: 심하게 손상되고 피투성이인 개체부터 깨끗한 개체까지 다양
- 움직임: 정지 시 마네킹처럼, 활성 시 유동적

**층별 특성**:
- **원작 인용**: "각 층에서 잘 벗어나지 않는다"
- **기존 기획안**: SYSTEM_BALANCING.md "층 격리 규칙"
- 학생들은 기본적으로 배정된 층에 머무름
- 1학년 학생 → 2층
- 2학년 학생 → 3층
- 3학년 학생 → 4층
- **예외**: 졸업식 트리거 (상세는 1.3.3 참조)

### 1.2 이동 및 배치

#### 층별 초기 배치

| 층 | 학생 유형 | 개체수 (대략) | 행동 패턴 |
|:---:|:---|:---:|:---|
| **1층** | 혼합/교직원 | 4-6개체 | 저밀도, 느린 순찰 |
| **2층** | 1학년 | 15-20개체 | 표준 추격, 중간 밀도 |
| **3층** | 2학년 | 15-20개체 | 표준 추격, 중간 밀도 |
| **4층** | 3학년 | **100개체 이상** | **구속된 팔 변종**, 극도로 높은 밀도 |
| **5층** | 졸업식장 | 300개 이상 (의자) | 식 시작 전까지 비활성 |

#### 4층 특수 메커니즘

**원작 증거**:
- "4층의 학생 개체들은 전원 3학년으로. 해당 층에 진입한 모든 탐사 회차에서 100구 이상의 개체수가 확인되었다."
- "복도에서 마네킹처럼 멈춘 채 사람을 응시하던 수십 명의 학생들"
- "발이 바닥에 묶인 채 11자로 늘어서 있음. 이동하지 못하고 팔만 허우적거림"

**4층 변종: "구속된 팔"**:
- **이동성**: 발이 바닥에 묶임 (발이 바닥에 묶임)
- **공격**: 3타일 반경 내에서만 팔을 휘두름
- **배치**: 복도 양옆으로 줄지어 서 있음
- **위협**: 통과해야 하는 가운틀렛 형성
- **손전등 효과**: 빛을 비추면 일시 정지

### 1.3 시나리오 진행에 따른 변화

> **기존 기획안 우선**: SYSTEM_BALANCING.md v2.0 Section 3.1 (점멸 시스템), Section 3.2 (묵념 시스템) 참조

#### 1.3.1 점멸/묵념 시스템 (동일 메커니즘)

**핵심**: 점멸과 묵념은 **기본적으로 같은 메커니즘**입니다. 둘 다 일정 시간 동안 불이 꺼지고, 플레이어가 학생 개체들을 볼 수 없게 되어 학생 개체들이 자유롭게 움직일 수 있으며, 플레이어는 숨어야만 공격을 받지 않을 수 있습니다.

**유일한 차이점**: 묵념은 사전 방송이 있음 (점멸은 방송 없이 발생)

---

**점멸 (Blackout) - 타이머 기반 자동 발생**

**트리거**: 층별 타이머 기반 (기존 기획안 기준)
- **기존 기획안 수치** (SYSTEM_BALANCING.md v2.0):
  - 1층: 420초 (7분)
  - 2층: 300초 (5분)
  - 3층: 240초 (4분)
  - 4층: 180초 (3분)
  - 5층: 점멸 없음 (보스전)

**메커니즘**:
1. **예고**: 5초 전 조명 깜빡임 (기존 기획안)
2. **암전**: 1초간 조명 완전 소등 (기존 기획안)
3. **재배치**: 학생 개체 순간이동 (플레이어 기준 3-10타일, 시야 밖)

**학생들에게 미치는 영향**:
- 암전 중 플레이어가 학생 개체를 볼 수 없음
- 학생들이 자유롭게 이동하여 새 위치로 순간이동
- 플레이어 근처로 재배치 가능
- **원작 인용**: "점멸 종료 - 세계 재배치. 불이 다시 켜짐. 학생 개체 위치 변화... 새로운 학생 C, D 출현 (시야 밖에서)"

**플레이어 대응**:
- 숨을 필요는 없음 (1초만 지속)
- 점멸 후 주변 경계 필수 (학생 재배치로 인한 포위 가능성)

---

**묵념 (Silent Prayer) - 학생 개체 처치 시 발생**

**트리거**: 학생 개체 사망 시 (기존 기획안 기준)
- **기존 기획안**: SYSTEM_BALANCING.md v2.0 Section 3.2
- 은신 처치 시에도 발동

**메커니즘** (기존 기획안):
1. **사망 확인 → 방송 대기**: 5초 (은신 준비 시간)
2. **방송**: "[방송] 잠시 후, 졸업생 故 이○○ 학생을 위한 묵념이 있겠습니다."
3. **묵념 지속시간**: 5초 (학교 종소리 "딩동댕동" 재생)
4. **암전**: 일반 모드는 70% 어두워짐, 전학생 모드는 150% 밝아짐

**학생들에게 미치는 영향**:
- **점멸과 동일**: 암전 중 플레이어가 학생들을 볼 수 없음
- 학생들이 자유롭게 이동 가능
- **기존 기획안**: 묵념 중 학생 속도 8.0 타일/초 (일반), 플레이어보다 빠름

**플레이어 대응** (기존 기획안):
- **일반 모드**: 교실/특수실/화장실에 숨어야 함 (복도/계단은 위험)
- **전학생 모드**: 묵념 중 밝기 150%, 플레이어 속도 9.0 타일/초 (달리기 속도)
- 5초 대기 시간 동안 은신 위치로 이동

**안전 구역 (묵념 중, 일반 모드)**:
- 교실 내부
- 특수실: 양호실, 도서실, 음악실, 창고 등
- 화장실
- **위험 구역**: 복도, 계단

---

**점멸 vs 묵념 비교표**

| 항목 | 점멸 (Blackout) | 묵념 (Silent Prayer) |
|:---|:---|:---|
| **트리거** | 층별 타이머 (자동) | 학생 개체 사망 시 |
| **사전 방송** | ❌ 없음 | ✅ 있음 (5초 대기 후) |
| **암전 지속** | 1초 | 5초 |
| **학생 이동** | ✅ 순간이동 (재배치) | ✅ 자유롭게 이동 |
| **플레이어 대응** | 경계 강화 | 은신 필수 (일반 모드) |
| **위험도** | 중간 (재배치) | 높음 (장시간 암전) |

---

#### 1.3.2 원작과의 차이점 (참고용)

**원작 묵념 장면**:
- **원작 인용**: "묵념. 모든 학생 개체 일제히 정지 (5초간)" / "복도를 거대한 그림자가 천천히 지나감 (선생님의 그림자)"
- 원작에서는 학생들이 정지하고 선생님의 그림자가 지나가는 연출

**게임 기획안 묵념**:
- **기존 기획안**: 학생들이 정지하는 것이 아니라 **암전 중 자유롭게 이동**
- 플레이어가 은신해야 살아남음
- 선생님 그림자 연출은 기획안에 명시되지 않음 (선택적 구현 가능)

**리서치 결론**: 게임 기획안의 묵념 시스템이 원작과 다르지만, 게임플레이 긴장감 조성을 위한 의도적 변경으로 판단됩니다. **기존 기획안 우선**하여 구현합니다.

#### 1.3.3 졸업식 (Graduation Ceremony)

**트리거**: 스토리 진행 / 5층 강당 진입 시

**기존 기획안 참조**: CHARACTERS_AND_ITEMS.md (선생님 트리거), SYSTEM_BALANCING.md (보스전)

---

**Phase 1: 학생 집결 방송**

**방송**:
> "[세광공업고등학교 학생 여러분! 졸업식이 곧 시작합니다. 강당으로 모여주세요.]"

**학생들에게 미치는 영향**:
- **모든 4층 3학년 학생들**이 5층 강당으로 이동
- 4층 복도가 일시적으로 비워짐
- 강당에 300개 이상의 개체 집결
- 학생들이 의자에 앉아 무대를 향함
- **원작 인용**: "복도를 빼곡히 메우고 있던 수많은 3학년 학생들은... (이동)"

**4층 상태 (일시적)**:
- 복도에서 3학년 학생 소멸
- 이동이 다소 용이해짐
- **그러나 잠깐뿐...**

---

**Phase 2: 선생님 추격 시작 (중요!)**

**방송**:
> "[참석하지 않은 학생에게 벌점을 주기 위해. 지금 선생님이 강당에서 출발합니다.]"

**원작 인용**: CHARACTERS_AND_ITEMS.md "빈 게임 케이스" 로어 텍스트

**선생님 개체 행동**:
- **5층 강당에서 출발**하여 아래 층으로 하강
- **사냥 모드 (Hunt Mode)** 활성화
- **추격 대상**: 플레이어 + 백일몽 주식회사 NPC + 재난관리국 NPC
- **이동 경로**: 5층 → 4층 → 3층 → 2층 → 1층 (순차 하강)
- **기존 기획안 스펙**:
  - 속도: 7.5 타일/초 (플레이어보다 빠름)
  - 공격력: 접촉 시 즉사
  - 시선 무효: 손전등으로 정지 불가

**위험도 변화**:
- Phase 1 직후: 4층이 일시적으로 안전 ✅
- Phase 2 방송 후: **전체 학교가 극도로 위험** ⚠️
  - 4층 학생 개체는 사라졌지만
  - 선생님이 5층에서 내려오며 전 층 사냥
  - **어떻게 보면 더 위험해진 상황**

**플레이어 대응**:
1. **도주**: 선생님을 피해 아래층으로 도망 (일시적)
2. **숨기**: 교실/특수실에 은신 (선생님이 지나갈 때까지)
3. **보스 직면** (엔딩 루트):
   - 해피엔딩 02: 5층 강당에서 40초 생존 (졸업식 완수)
   - 배드엔딩 04: 선생님에게 사망 (학교의 종말)

**기존 기획안 연결**:
- **선생님 트리거**: CHARACTERS_AND_ITEMS.md "선생님" 섹션
  - 트리거 1: 명찰 4개 이상 소지
  - 트리거 2: 졸업식 시작 (5층 강당 진입)
- **보스전**: SYSTEM_BALANCING.md Section 2.4 / GAME_DESIGN_SAEKWANG_HIGHSCHOOL.md Section 3.9.6

---

**졸업식 타임라인 요약**

```
00:05 - 4층 3학년 학생들 강당으로 이동 시작
00:10 - 4층 복도 비워짐 (일시적 안전)

00:15 - Phase 2 방송 "[선생님이 강당에서 출발합니다.]"
00:16 - 선생님 개체 5층에서 사냥 시작
00:20 - 선생님 4층 도착 (플레이어 추격)
...   - 선생님 계속 하강 (5→4→3→2→1층)

[엔딩 분기]
- 5층 강당 진입 → 보스전 (40초 생존) → 해피엔딩 02

```

**리서치 결론**: 졸업식은 단순히 "4층이 안전해지는" 이벤트가 아니라, **선생님 개체의 전역 추격을 트리거하여 게임 전체의 위험도를 극대화하는 클라이맥스 이벤트**입니다. 기존 기획안의 의도를 정확히 반영합니다.

### 1.4 플레이어 상호작용 메커니즘

#### 일반 모드 (Normal Mode)

**플레이어 인식**:
- 학생들이 얼굴 없는 적대적 개체로 보임
- 모든 학생이 위협
- 생존을 위해 손전등 + 시선 유지 필요
- 학생들과 소통 불가

**플레이어에 대한 학생 행동**:
- 관찰되지 않을 때 적대적 추격
- 관찰될 때 정지
- 접촉 시 공격 (50-100 데미지)
- 대화나 상호작용 불가능

**원작 인용**: "시선이 없어지는 순간 [학생들이] 즉시 이동 시작 (1.5 tiles/s)"

#### 전학생 모드 (Transfer Student Mode)

**활성화 조건**:
- 이결의 교복 획득
- 최 요원과의 양호실 루트를 통해 트리거

**원작 증거**:
- "이결에게 교복을 빌렸던 1학년 5반의 전학생으로"
- "넌 진짜 전학생은 아니잖아. 김솔음."

**플레이어 인식 - 반전**:
- 학생들이 정상적이고 살아있는 급우로 보임
- 말하고 (무음), 몸짓하고, 서로 상호작용함
- 표정, 감정, 사회적 행동을 보임
- **원작 인용**: "학생들은 이제 생동감 있는 표정으로 서로 눈짓을 주고받거나 손짓을 하면서 서로 소리 없는 잡담을 나누고 있다."

**중요 규칙**:
- **원작 인용**: "세광공업고등학교의 학생들에겐 목소리가 없다."
- 학생들은 몸짓, 문자메시지, 수화로 소통
- 음성 대화 없음

**전학생에 대한 학생 행동**:
- 비적대적 (도발하지 않는 한)
- 몸짓으로 힌트 제공 가능
- 안전하게 접근 가능
- 함께 앉고, 아이템 제공하고, 플레이어 안내
- **원작 인용**: "대여섯 명의 학생들이 모여서 둘러앉아 있다... 학생들이 의아한 표정으로 나를 보았다."

**관점 전환**:
- **일반 모드**: "괴물로부터 탈출"
- **전학생 모드**: "갇힌 급우들 사이에서 생활"

### 1.5 명찰 시스템

**핵심 게임플레이 아이템**

**기능**:
- **일반 사망**: 명찰을 들고 죽음 → 현실 세계에서 안전하게 깨어남
- **명찰 없이 사망**: 괴담에 영구히 묶임, 매월 그믐날 끌려옴

**원작 인용**:
- "저 인형의 명찰을 가진 채로 죽어야만 '정상적으로' 깨어날 수 있습니다."
- "명찰이 없어 죽으면 무슨 일이 벌어지는지 아십니까?"
- "학생의 명찰을 뺏는 것을 목격당할 시. 해당 학생에게 죽을 때까지 추격당한다."

**획득 규칙**:
1. **은밀함 필요**: 명찰을 가져가는 것을 해당 학생이 관찰하지 않아야 함
2. **목격 패널티**: 학생이 플레이어가 자신의 태그를 가져가는 것을 "보면" → 무한 추격
3. **손전등 기술**: 빛으로 학생을 정지시키고, 뒤에서 접근하여 태그 제거

**층별 명찰 개수**:
- 시나리오에 따라 다름
- 일부 층은 4개 이상 접근 가능한 태그 보유
- 4층은 많지만 수집하기 극도로 위험

**선생님 트리거 (Teacher Trigger)**:
- **4개 이상 명찰 소지** → 선생님이 5층에서 추격 시작
- 선생님은 태그 개수에 따라 플레이어 위치 추적
  - 4개: 완벽한 추적
  - 3개: 대략적인 위치
  - 2개: 약화된 추적
  - 1개: 최소 추적

**게임 구현 권장사항**:
- **라이벌 NPC**: 산발적으로 나타나 긴장감 조성

- 엔딩 분기: 플레이어가 도움 또는 포기 선택

**상호작용 포인트**:

1. **기숙사 장면 (진입 전)**:
   - 백사헌이 플레이어 방에 초대 없이 들어옴

   - 힌트 주면: 백사헌 진입 (나중에 나타남)

2. **2층 복도 조우 (세광 내부)**:

     - [무시]: 백사헌 사망 또는 별도 탈출
   - 엔딩 가용성에 영향

**대화 특성**:

- 신뢰 확립까지 플레이어 이름 거의 사용 안 함
- 가끔 건조한 유머
- 원칙적이지만 잔인하지 않음

**협력 보상**:
- 배터리 (손전등 사용 연장)
- 전술 조언 (안전 경로, 점멸 타이밍)
- 주요 순간 백업 (4층 가운틀렛, 졸업식)
- "재난관리국 협력" 엔딩 변형 해금

**비주얼 디자인**:
- 키 크고 어깨 넓은 실루엣
- 어두운 재난관리국 유니폼 (교복 아님)
- 뚜렷한 눈 밑 그림자, 피곤한 표정
- 날카롭고 분석적인 시선
- 손전등 + 부적 주머니 휴대

#### 3.2.2 최 요원 - [로어/맥락만]

**소속**: 초자연 재난관리국
**직책**: 대응부 에이스 작전요원
**팀**: 현무 1팀

**외형**:
- 남성, 키 크고 단정한 옷차림
- 인상적인 푸른 눈
- 뚜렷한 목 흉터 (냉동고 관련 사건, PTSD 유발)

**세광 아크 관련성**:
- **주요 세광 시나리오에 직접 등장하지 않음**
- **전학생 루트에 등장** (해피엔딩 01 경로)
- 양호실 이벤트와 연결
- 이결의 교복 획득 촉진

**원작 증거 (전학생 맥락)**:
- "부상당한 학생 개체가 적절한 구급처치를 받은 채 체육복 차림으로 침대에 누워 있는 것을 목격"
- 부상당한 학생 개체 치료 (양호실의 최 요원)
- 플레이어가 대체 교복 획득 방법과 연결

**게임 구현**:
- **고급 루트용 선택 NPC**
- 조건: 플레이어가 양호실 진입 + 의료 보급품 보유 시 등장
- 이결 연결 경로 제공
- **기본 게임 완료에 필수 아님**

**로어 중요성**:
- 재난관리국 요원들이 세광에 갇힌 이유 설명
- 구조 임무 긴급성 맥락 제공
- 재난관리국/회사 갈등 심화 (회사가 민간인 버림, 재난관리국 구조)

#### 3.2.3 박홍림 - [중요 로어]

**소속**: 초자연 재난관리국 (이전)
**직책**: 팀장, 현무 1팀
**코드명**: 홍화

**현재 상태**: **"선생님" 개체로 변형됨**

**외형 (사건 전)**:
- 여성, 노년
- 지혜롭고 친절한 인상

**현재 형태 (선생님으로)**:
- 이전 재해에서 상체 손실
- 오로라 같은 길상의 영적 빛으로 나타남
- 현무 1팀 대기실 창문 너머 커튼 뒤에서 드러남
- 꿈 상태로 입원
- **추정**: 원래 형태 상실, 세광특별시에 반신 남겨둔 것과 유사

**중요 로어 연결**:
- **소설/웹 리서치 증거**: "박홍림 팀장/홍화 (Honghwa) - 선생님 개체로 변형"
- 박홍림의 의식/신체 손상 → 세광의 선생님이 됨
- 선생님의 기원 설명 (원래 괴담 요소가 아니라 손상된 재난관리국 요원)
- **선생님이 나타나는 이유**: 박이 구조 임무 시도, 갇히고/변형됨

**능력 (손상 전)**:
- 도깨비불 호롱 (도깨비 불 등)
- 도깨비 장난 (도깨비 능력)
- 요원들을 어린이로 변형 가능 (신체 변형)

**게임 구현**:
- **로어 드러남**: 플레이어가 문서를 통해 발견 가능
- 숨겨진 로어: 5층 사무실에서 박의 재난관리국 ID 찾기
- 류의 대화 힌트: "우리 팀장님이... 몇 달 전 여기 오셨어. 돌아오지 못했어."
- **감정적 무게**: 보스전이 비극적 맥락 (손상된 영웅, 순수 악이 아님)
- **해피엔딩 01**: 박의 의식 해방이 해결의 일부

**게임플레이 연결**:
- 선생님이 "도깨비" 요소 사용하는 이유 설명 (도깨비불 모티프)
- 선생님이 전술 지능을 가진 이유 (이전 재난관리국 팀장)
- 명찰 4개 이상이 선생님을 트리거하는 이유 (박의 손상된 보호 본능 → 과도 수집자 사냥으로 왜곡)

### 3.3 세광에서 재난관리국 vs 회사 역학

**조직 갈등**:
- **원작 인용**: "내 앞에서 지나가는 우리 회사 직원들을 경계와 멸시가 어린 눈으로 쳐다보는 공무원"
- 재난관리국은 회사를 비윤리적 착취자로 봄
- 회사는 재난관리국을 이익에 대한 간섭으로 봄
- **프로토콜**: 재난관리국이 공무집행방해로 회사 직원 체포

**세광 아크에서**:
- 재난관리국 임무: 갇힌 민간인 구조 + 팀장(박홍림) 회수
- 회사 임무: 꿈결 수확, 장비 테스트, 데이터 수집
- **강제 협력**: 김솔음이 류와의 간극 메움
- **원작 인용**: "그렇게 백일몽 주식회사 직원과 재난관리국 공무원은 급박히 한 사람을 부축해 교실 안으로 들어왔다"
- 적대에도 불구하고 위기 시 공유된 인간성이 협력으로 이끔

**게임 구현**:
- 선택 시스템: 재난관리국 정렬 (윤리적) vs 회사 (실용적)
- 류의 협력이 재난관리국 정렬 엔딩 해금
- 백사헌 협력이 회사 정렬 엔딩 해금
- 중립 경로: 둘 다 사용 (최선의 결과, 가장 어려움)

### 7.2 우선순위 2: 시나리오 이벤트

1. **점멸 시스템**:

   - 무작위 트리거 시스템 (층 전환당 15-25%)
   - PA 안내방송 오디오
   - 5초 카운트다운 UI

   - 은신 메커니즘 ([Ctrl] 키, 1인칭 카메라)

3. **졸업식**:
   - 스토리 플래그 트리거 + 5층 접근 트리거
   - PA 안내방송
   - 4층 개체 대규모 이동 경로 찾기
   - 5층 강당 좌석 배치
